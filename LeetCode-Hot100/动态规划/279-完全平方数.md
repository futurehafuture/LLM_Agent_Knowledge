# 279. 完全平方数

> 链接：https://leetcode.cn/problems/perfect-squares/

## 题目描述

给整数 `n`，返回和为 `n` 的**最少完全平方数**数量。完全平方数可以重复使用。

### 示例

```
输入：n = 12
输出：3
解释：12 = 4 + 4 + 4
```

```
输入：n = 13
输出：2
解释：13 = 4 + 9
```

约束：`1 <= n <= 10^4`

---

## 思路推导（DP 五步法）

### ① 定义状态

`dp[j]` = 凑出 `j` 所需的最少完全平方数个数。

### ② 写出转移方程

物品就是 $`1^2, 2^2, 3^2, \ldots, \lfloor\sqrt{j}\rfloor^2`$：

```math
dp[j] = \min_{1 \le i \le \lfloor\sqrt{j}\rfloor} (dp[j],\; dp[j - i^2] + 1)
```

和 [[322-零钱兑换]] 完全一样的模型——物品变成完全平方数而已。

### ③ 初始化

- `dp[0] = 0`
- `dp[j] = ∞`（`j > 0`）

### ④ 遍历顺序

外层 `i` 从 1 到 `⌊√n⌋`，内层 `j` 从 `i²` 到 `n` **正序**（完全背包）。

### ⑤ 返回值

`dp[n]`。

## Python 代码

### 写法一：完全背包 DP

```python
import math

class Solution:
    def numSquares(self, n: int) -> int:
        dp = [float('inf')] * (n + 1)
        dp[0] = 0
        for i in range(1, int(math.isqrt(n)) + 1):
            square = i * i
            for j in range(square, n + 1):
                dp[j] = min(dp[j], dp[j - square] + 1)
        return dp[n]
```

### 写法二：外层容量版本

```python
class Solution:
    def numSquares(self, n: int) -> int:
        dp = [float('inf')] * (n + 1)
        dp[0] = 0
        for j in range(1, n + 1):
            i = 1
            while i * i <= j:
                dp[j] = min(dp[j], dp[j - i * i] + 1)
                i += 1
        return dp[n]
```

> [!question] 这个写法为什么能保证完全平方数被重复使用？
>
> 关键在于 `dp[j - i*i]` 是在**更小的 j** 时已经算好的，而那个值本身可能就是用 `i*i` 凑出来的。
>
> 以 `n=12` 为例推演：
>
> | j | 检查的 i | dp[j] 怎么来的 | 含义 |
> |---|---|---|---|
> | 4 | i=1→dp[3]+1=inf, i=2→dp[0]+1=1 | dp[4]=1 | 4 = 2² |
> | 8 | i=2→dp[4]+1=1+1=2 | dp[8]=2 | 8 = 2² + 2² |
> | 12 | i=2→dp[8]+1=2+1=3 | dp[12]=3 | 12 = 2² + 2² + 2² |
>
> 算 `dp[8]` 时，`dp[8-4]=dp[4]=1`（用了一个 2²），再加 1 表示又用了一个 2²，所以总共用了两次 2²。
>
> **本质**：外层按容量 `j` 遍历，每个 `dp[j]` 的计算不限制用了哪些数、用了几次。`dp[j-i*i]` 的值里可能已经包含了 `i*i`，再加一个 `i*i` 就等于"重复使用"。不需要任何额外机制来保证可重复——它天然就是可重复的。
>
> 对比写法一：写法一是"逐个物品放入"，需要正序遍历来允许重复；写法二是"逐个容量去凑"，内层试所有可能的最后一个数，`dp` 的子问题天然包含了重复使用的可能性。

### 写法三：BFS

```python
from collections import deque
import math

class Solution:
    def numSquares(self, n: int) -> int:
        squares = [i * i for i in range(1, int(math.isqrt(n)) + 1)]
        visited = set()
        queue = deque([(n, 0)])    # (剩余值, 步数)
        visited.add(n)
        while queue:
            remaining, steps = queue.popleft()
            for sq in squares:
                nxt = remaining - sq
                if nxt == 0:
                    return steps + 1
                if nxt < 0:
                    break
                if nxt not in visited:
                    visited.add(nxt)
                    queue.append((nxt, steps + 1))
        return n    # 最坏全是 1
```

### 写法四：数学（四平方和定理）

```python
import math

class Solution:
    def numSquares(self, n: int) -> int:
        # 检查是否是完全平方数
        if int(math.isqrt(n)) ** 2 == n:
            return 1
        # 检查是否是两个平方数之和
        for i in range(1, int(math.isqrt(n)) + 1):
            if int(math.isqrt(n - i * i)) ** 2 == n - i * i:
                return 2
        # 检查是否满足 4^a(8b+7) 形式
        temp = n
        while temp % 4 == 0:
            temp //= 4
        if temp % 8 == 7:
            return 4
        return 3
```

## 易错点

1. **DP 初始化用 `inf`**：和 322 一样，求最小值初始必须为无穷大。
2. **`dp[0] = 0` 不能忘**：这是递推的起点。
3. **数学解法的判断顺序**：先判 1，再判 2，再判 4（通过 4^a(8b+7)），最后才是 3。

## 知识点补充

### 四平方和定理（Lagrange, 1770）

任意正整数都可以表示为至多 4 个完全平方数之和。因此答案只有 1、2、3、4 四种可能。

Legendre 进一步给出了判断规则：
- 答案为 1 ↔ n 本身是完全平方数
- 答案为 4 ↔ n = $`4^a(8b+7)`$
- 答案为 2 ↔ n = $`a^2 + b^2`$
- 其余答案为 3
