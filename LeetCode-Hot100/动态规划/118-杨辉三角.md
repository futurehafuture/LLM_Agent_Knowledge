# 118. 杨辉三角

> 链接：https://leetcode.cn/problems/pascals-triangle/

## 题目描述

给定一个非负整数 `numRows`，生成「杨辉三角」的前 `numRows` 行。在杨辉三角中，**每个数是它左上方和右上方的数之和**。

### 示例

```
输入：numRows = 5
输出：
[
     [1],
    [1,1],
   [1,2,1],
  [1,3,3,1],
 [1,4,6,4,1]
]
```

```
输入：numRows = 1
输出：[[1]]
```

约束：`1 <= numRows <= 30`

---

## 思路推导

### ① 定义状态

`dp[i][j]` = 杨辉三角第 i 行第 j 列的值（`i, j` 都从 0 开始）。

### ② 转移方程

```math
dp[i][j] =
\begin{cases}
1 & j = 0 \text{ 或 } j = i \quad\text{（每行首尾）} \\
dp[i-1][j-1] + dp[i-1][j] & \text{其他情况}
\end{cases}
```

### ③ 初始化

`dp[0] = [1]`。

### ④ 遍历顺序

`i` 从 0 到 `numRows-1` 正向，`j` 从 0 到 `i` 正向。

### ⑤ 返回值

整个 `dp` 数组。

---

## Python 代码

### 写法一：标准二维 DP

```python
class Solution:
    def generate(self, numRows: int) -> list[list[int]]:
        triangle = []
        for i in range(numRows):
            row = [1] * (i + 1)                 # 先全填 1，搞定首尾
            for j in range(1, i):                # 中间元素由上一行算出
                row[j] = triangle[i-1][j-1] + triangle[i-1][j]
            triangle.append(row)
        return triangle
```

### 写法二：错位相加（更 Pythonic）

```python
class Solution:
    def generate(self, numRows: int) -> list[list[int]]:
        triangle = [[1]]
        for _ in range(numRows - 1):
            prev = triangle[-1]
            new_row = [1] + [a + b for a, b in zip(prev, prev[1:])] + [1]
            triangle.append(new_row)
        return triangle
```

> 这里用 `zip(prev, prev[1:])` 把前一行错位相加得到中间元素，再两端各补一个 1。

## 复杂度分析

| 项 | 复杂度 |
|---|---|
| 时间 | O(numRows²)：第 i 行有 i+1 个元素，总和 ≈ n²/2 |
| 空间 | O(numRows²)：返回结果本身就这么大，没法压缩 |

如果题目只要求**第 k 行**（LeetCode 119），可以用 O(k) 空间——见下方相关题目。

---

## 易错点

1. **`row = [1] * (i+1)`** 而不是 `(i)`——第 i 行有 i+1 个元素（0-indexed）。
2. **内层循环 `range(1, i)`** 不是 `range(1, i+1)`——首尾的 1 已经填好，只算中间。
3. **`triangle[i-1][j-1] + triangle[i-1][j]`**——很多人会写成 `[j-1] + [j+1]`，记住是"左上 + 正上"。

## 知识点补充

### 杨辉三角 = 二项式系数

第 i 行第 j 列恰好是 $`\binom{i}{j}`$：

```math
\binom{i}{j} = \binom{i-1}{j-1} + \binom{i-1}{j}
```

这正是组合数的递推公式（**帕斯卡恒等式**）。也可以直接用：

```math
\binom{i}{j} = \frac{i!}{j!(i-j)!}
```

### 错位相加技巧

`[a + b for a, b in zip(prev, prev[1:])]` 是个常用 Python 技巧，把"相邻元素两两相加"用一行表达。同样可以写差分、前缀和。

### 滚动一行的写法（LeetCode 119）

如果只要第 k 行，逆向更新就能 O(k) 空间：

```python
def getRow(rowIndex: int) -> list[int]:
    row = [1] * (rowIndex + 1)
    for i in range(1, rowIndex + 1):
        for j in range(i - 1, 0, -1):           # 倒序避免覆盖未读的左边
            row[j] = row[j] + row[j - 1]
    return row
```

> **倒序**的原因：和 0/1 背包一样，正序会让本轮已更新的值覆盖掉本来该读的"上一行"值。
