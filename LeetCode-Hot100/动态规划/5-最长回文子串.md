# 5. 最长回文子串

> 链接：https://leetcode.cn/problems/longest-palindromic-substring/

## 题目描述

字符串 `s`，找到 `s` 中最长的回文子串。

### 示例

```
输入：s = "babad"
输出："bab"
解释："aba" 同样是合法答案。
```

```
输入：s = "cbbd"
输出："bb"
```

约束：`1 <= s.length <= 1000`

---

## 思路推导（DP 五步法）

### ① 定义状态

`dp[i][j]` = 子串 `s[i..j]` 是否为回文（布尔值）。

### ② 写出转移方程

`s[i..j]` 是回文，需要满足：

1. 首尾字符相同：`s[i] == s[j]`
2. 中间部分也是回文：`dp[i+1][j-1] == True`（或子串长度 ≤ 3 时自动满足）

```math
dp[i][j] =
\begin{cases}
True & \text{if } s[i] = s[j] \text{ and } (j - i < 3 \text{ or } dp[i+1][j-1]) \\
False & \text{otherwise}
\end{cases}
```

> `j - i < 3` 的含义：子串长度 ≤ 3 时（如 `aa`、`aba`），只要首尾相同就是回文，不需要检查中间。

### ③ 初始化

`dp[i][i] = True`（单个字符是回文）。

### ④ 遍历顺序

关键！`dp[i][j]` 依赖 `dp[i+1][j-1]`（左下方），所以必须**先算短的子串，再算长的**。

两种方式：
- 外层按子串长度 `length` 从 2 到 `n`
- 外层 `i` 从 `n-1` 到 `0`（反向），内层 `j` 从 `i+1` 到 `n-1`

### ⑤ 返回值

遍历过程中记录最长回文的起始索引和长度。

## Python 代码

### 写法一：区间 DP（按长度遍历）

```python
class Solution:
    def longestPalindrome(self, s: str) -> str:
        n = len(s)
        if n <= 1:
            return s

        dp = [[False] * n for _ in range(n)]
        start, max_len = 0, 1

        for i in range(n):
            dp[i][i] = True

        for length in range(2, n + 1):
            for i in range(n - length + 1):
                j = i + length - 1
                if s[i] == s[j]:
                    if length <= 3:
                        dp[i][j] = True
                    else:
                        dp[i][j] = dp[i+1][j-1]
                if dp[i][j] and length > max_len:
                    start, max_len = i, length

        return s[start:start + max_len]
```

### 写法二：中心扩展法（O(1) 空间）

从每个位置向两边扩展，分别处理奇数长度和偶数长度。

```python
class Solution:
    def longestPalindrome(self, s: str) -> str:
        n = len(s)
        if n <= 1:
            return s

        start, max_len = 0, 1

        def expand(left: int, right: int) -> tuple[int, int]:
            while left >= 0 and right < n and s[left] == s[right]:
                left -= 1
                right += 1
            # 退出时 left, right 已经越界或不相等
            return left + 1, right - left - 1

        for i in range(n):
            # 奇数长度：以 i 为中心
            s1, l1 = expand(i, i)
            # 偶数长度：以 i, i+1 为中心
            s2, l2 = expand(i, i + 1)

            s_best, l_best = (s1, l1) if l1 >= l2 else (s2, l2)
            if l_best > max_len:
                start, max_len = s_best, l_best

        return s[start:start + max_len]
```

### 写法三：Manacher 算法（O(n)）

面试一般不要求，了解即可。

```python
class Solution:
    def longestPalindrome(self, s: str) -> str:
        # 预处理：插入分隔符，统一处理奇偶
        t = '#' + '#'.join(s) + '#'
        n = len(t)
        p = [0] * n        # p[i] = 以 i 为中心的最长回文半径
        center = right = 0
        max_center = 0

        for i in range(n):
            if i < right:
                mirror = 2 * center - i
                p[i] = min(right - i, p[mirror])

            # 尝试扩展
            while (i - p[i] - 1 >= 0 and i + p[i] + 1 < n
                   and t[i - p[i] - 1] == t[i + p[i] + 1]):
                p[i] += 1

            # 更新中心和右边界
            if i + p[i] > right:
                center, right = i, i + p[i]

            if p[i] > p[max_center]:
                max_center = i

        start = (max_center - p[max_center]) // 2
        return s[start:start + p[max_center]]
```

## 复杂度分析

| 写法 | 时间 | 空间 |
|---|---|---|
| 区间 DP | O(n²) | O(n²) |
| 中心扩展 | O(n²) | O(1) |
| Manacher | O(n) | O(n) |

---

## 易错点

1. **遍历顺序**：`dp[i][j]` 依赖 `dp[i+1][j-1]`（左下方），不能简单地 `i` 从小到大、`j` 从小到大，否则会用到还没计算的值。按子串长度遍历最安全。
2. **`j - i < 3` 的边界**：长度为 2 时（`j - i = 1`），只要 `s[i] == s[j]` 就是回文。长度为 3 时（`j - i = 2`），同样首尾相同即可。
3. **中心扩展的返回值**：退出 while 时 `left` 和 `right` 已经越界一步，所以实际回文范围是 `[left+1, right-1]`。
4. **`dp[i][i] = True`**：单个字符一定是回文。

## 知识点补充

### 区间 DP

本题是经典的区间 DP：状态 `dp[i][j]` 表示**区间 [i,j]** 的答案，转移时区间逐渐扩大。常见模式：

| 题目 | 状态 | 代表 |
|---|---|---|
| 5 最长回文子串 | `dp[i][j]` = 是否回文 | 区间判断 |
| 312 戳气球 | `dp[i][j]` = 区间最大得分 | 区间最优化 |

### 中心扩展法的直觉

回文有"对称性"，可以从中心向两边扩展。每个位置有两种中心：
- 单字符中心（奇数长度回文）
- 双字符中心（偶数长度回文）

共 2n-1 个中心，每个中心最多扩展 O(n) 次，总 O(n²)。

### Manacher 算法的核心思想

利用回文的对称性，避免重复扩展。维护当前最右回文的中心和边界，对每个位置利用对称性"免费"获得一部分回文半径，再尝试扩展。
