# 70. 爬楼梯

> 链接：https://leetcode.cn/problems/climbing-stairs/

## 题目描述

假设你正在爬楼梯。需要 `n` 阶你才能到达楼顶。每次你可以爬 **1 或 2 个台阶**。求有多少种**不同的方法**可以爬到楼顶？

### 示例

```
输入：n = 3
输出：3
解释：1+1+1 / 1+2 / 2+1
```

```
输入：n = 5
输出：8
```

约束：`1 <= n <= 45`

---

## 思路推导（DP 五步法）

### ① 定义状态

`dp[i]` = 爬到第 `i` 阶的不同方法数。

### ② 写出转移方程

要到第 `i` 阶，最后一步只有两种可能：
- 从第 `i-1` 阶爬 1 步上来 → `dp[i-1]` 种走法
- 从第 `i-2` 阶爬 2 步上来 → `dp[i-2]` 种走法

两种来源不重叠，所以：

```math
dp[i] = dp[i-1] + dp[i-2]
```

> 这就是著名的**斐波那契数列**递推。

### ③ 初始化

- `dp[0] = 1`（爬 0 阶有 1 种方法：原地不动）
- `dp[1] = 1`（爬 1 阶只有 1 种方法）

### ④ 遍历顺序

`i` 从 2 到 n 正向遍历。

### ⑤ 返回值

`dp[n]`。

## Python 代码

### 写法一：朴素 DP（数组版）

```python
class Solution:
    def climbStairs(self, n: int) -> int:
        if n <= 1:
            return 1
        dp = [0] * (n + 1)
        dp[0], dp[1] = 1, 1
        for i in range(2, n + 1):
            dp[i] = dp[i-1] + dp[i-2]
        return dp[n]
```

### 写法二：滚动数组（空间最优）

只依赖前两个状态，没必要存整个数组。

```python
class Solution:
    def climbStairs(self, n: int) -> int:
        prev, curr = 1, 1                       # dp[0], dp[1]
        for _ in range(2, n + 1):
            prev, curr = curr, prev + curr      # 同时赋值，否则会先覆盖
        return curr
```

### 写法三：记忆化搜索（自顶向下）

```python
from functools import cache

class Solution:
    def climbStairs(self, n: int) -> int:
        @cache
        def f(i: int) -> int:
            if i <= 1:
                return 1
            return f(i-1) + f(i-2)
        return f(n)
```

## 复杂度分析

| 写法 | 时间 | 空间 |
|---|---|---|
| 朴素 DP | O(n) | O(n) |
| 滚动数组 | O(n) | O(1) |
| 记忆化搜索 | O(n) | O(n)（递归栈+cache） |

也存在 O(log n) 的**矩阵快速幂**或**通项公式**解法，面试不要求。

---

## 易错点

1. **`prev, curr = curr, prev + curr` 必须同时赋值**。如果分两行：
   ```python
   prev = curr          # ❌ prev 被覆盖了
   curr = prev + curr   # 用的是新 prev，结果错
   ```
2. **`n=1` 边界**：朴素写法里如果直接 `dp[1]` 而 n=1，Range(2, 2) 不进循环，返回 `dp[1]=1` 是对的——但要确保 `dp` 数组至少有 2 个位置。
3. **`dp[0]=1` 不是 0**：可以理解为"还没动"也是一种"到达 0 阶"的方法，是为了让 `dp[2] = dp[1] + dp[0] = 2` 算对。

## 知识点补充

### 斐波那契数列

爬楼梯本质上就是斐波那契数列 $`F_n = F_{n-1} + F_{n-2}`$，只不过初值不同：
- 标准斐波那契：$`F_0=0, F_1=1`$
- 爬楼梯：$`f_0=1, f_1=1`$（其实是斐波那契右移一位）

### 进阶：每次能爬 1, 2, 3 阶？

转移变成：
```math
dp[i] = dp[i-1] + dp[i-2] + dp[i-3]
```
泰波那契数列。**思维框架不变**。

### 进阶：每次能爬任意 1 ~ k 阶？

```math
dp[i] = \sum_{j=1}^{k} dp[i-j]
```
本质等价于"凑出 i 的方法数"，和**完全背包求方案数**等价（先物品后容量）。
