# 32. 最长有效括号

> 链接：https://leetcode.cn/problems/longest-valid-parentheses/

## 题目描述

给一个只含 `(` 和 `)` 的字符串，找出最长有效（格式正确）**连续**括号子串的长度。

### 示例

```
输入：s = "(()"
输出：2
解释：最长有效括号子串是 "()"
```

```
输入：s = ")()())"
输出：4
解释：最长有效括号子串是 "()()"
```

```
输入：s = ""
输出：0
```

约束：`0 <= s.length <= 3*10^4`

---

## 思路推导（DP 五步法）

### ① 定义状态

`dp[i]` = 以 `s[i]` **结尾**的最长有效括号子串的**长度**。

> 为什么是"以 i 结尾"？因为有效括号一定以 `)` 结尾，这样只需考虑 `s[i] = ')'` 的情况。

### ② 写出转移方程

当 `s[i] = ')'` 时，分两种情况：

**情况一**：`s[i-1] = '('`（和 `s[i]` 配成一对 `()`）

```math
dp[i] = dp[i-2] + 2
```

即在 `s[i-1]` 之前的有效长度上加 2。

**情况二**：`s[i-1] = ')'`（`s[i]` 和前面的某个 `(` 配对）

如果 `s[i-1]` 本身就是某个有效子串的结尾，那 `s[i]` 应该和 `s[i - dp[i-1] - 1]` 配对：

```math
\text{若 } s[i - dp[i-1] - 1] = \text{'('}: \quad dp[i] = dp[i-1] + 2 + dp[i - dp[i-1] - 2]
```

这三部分分别是：`s[i-1]` 对应的有效长度 + 新配对的 2 + 再前面的有效长度。

### ③ 初始化

`dp[i] = 0` for all i（默认无效）。

### ④ 遍历顺序

`i` 从 1 到 `n-1` 正向。

### ⑤ 返回值

`max(dp)`。

## Python 代码

### 写法一：DP

```python
class Solution:
    def longestValidParentheses(self, s: str) -> int:
        n = len(s)
        dp = [0] * n
        for i in range(1, n):
            if s[i] == ')':
                if s[i-1] == '(':
                    # 情况一：...()
                    dp[i] = (dp[i-2] if i >= 2 else 0) + 2
                elif i - dp[i-1] - 1 >= 0 and s[i - dp[i-1] - 1] == '(':
                    # 情况二：...))
                    prev = dp[i - dp[i-1] - 2] if i - dp[i-1] >= 2 else 0
                    dp[i] = dp[i-1] + 2 + prev
        return max(dp) if dp else 0
```

### 写法二：栈

栈中保存**索引**，不匹配的 `)` 作为"分割点"。

```python
class Solution:
    def longestValidParentheses(self, s: str) -> int:
        stack = [-1]              # 初始边界
        max_len = 0
        for i, ch in enumerate(s):
            if ch == '(':
                stack.append(i)
            else:
                stack.pop()
                if not stack:
                    stack.append(i)   # 当前 ) 作为新边界
                else:
                    max_len = max(max_len, i - stack[-1])
        return max_len
```

### 写法三：双向扫描（O(1) 空间）

```python
class Solution:
    def longestValidParentheses(self, s: str) -> int:
        max_len = 0

        # 从左到右
        left = right = 0
        for ch in s:
            if ch == '(':
                left += 1
            else:
                right += 1
            if left == right:
                max_len = max(max_len, 2 * right)
            elif right > left:
                left = right = 0

        # 从右到左（处理 (() 这种情况）
        left = right = 0
        for ch in reversed(s):
            if ch == '(':
                left += 1
            else:
                right += 1
            if left == right:
                max_len = max(max_len, 2 * left)
            elif left > right:
                left = right = 0

        return max_len
```

## 复杂度分析

| 写法 | 时间 | 空间 |
|---|---|---|
| DP | O(n) | O(n) |
| 栈 | O(n) | O(n) |
| 双向扫描 | O(n) | O(1) |

---

## 易错点

1. **DP 情况二的索引**：`i - dp[i-1] - 1` 要检查 `>= 0`，否则越界。
2. **DP 情况二还要加上 `dp[i - dp[i-1] - 2]`**：配对之后，前面可能还有一段连续的有效括号。
3. **栈方法中 `stack.pop()` 前没检查空**：初始化放了 `-1` 保证第一次 pop 不会空。
4. **双向扫描为什么要扫两遍**：从左到右 `left > right` 时不会 reset，但从右到左会。比如 `(()` 只有从右到左才能正确计算。

## 知识点补充

### DP 转移的核心图解

```
... (  ...有效...  )
    ^              ^
    j              i

j = i - dp[i-1] - 1
如果 s[j] == '('：
dp[i] = dp[i-1] + 2 + dp[j-1]
        ↑内部有效↑   ↑配对↑  ↑前面可能还有↑
```

### 栈方法的本质

栈存的是"尚未匹配的 `(` 的位置"加上"分割边界"。两两匹配之间的距离就是有效长度。

### 三种方法的取舍

- **面试推荐**：栈方法，思路清晰，代码短。
- **空间最优**：双向扫描，O(1) 空间。
- **理解 DP**：DP 写法虽然复杂，但思路适用于更多括号类题目。
