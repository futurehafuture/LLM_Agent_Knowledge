# 198. 打家劫舍

> 链接：https://leetcode.cn/problems/house-robber/

## 题目描述

沿街有一排房屋，每个房屋内有一定金额。**不能偷相邻的两家**，求能偷到的最大金额。

### 示例

```
输入：nums = [1,2,3,1]
输出：4
解释：偷第 1 家（金额=1）和第 3 家（金额=3），总共 1+3=4。
```

```
输入：nums = [2,7,9,3,1]
输出：12
解释：偷第 1 家（2）、第 3 家（9）、第 5 家（1），总共 2+9+1=12。
```

约束：`1 <= nums.length <= 100`，`0 <= nums[i] <= 400`

---

## 思路推导（DP 五步法）

### ① 定义状态

`dp[i]` = 考虑前 `i` 个房屋（`nums[0..i-1]`）能偷到的最大金额。

### ② 写出转移方程

对第 `i` 个房屋（`nums[i-1]`），有两种选择：

- **不偷**：最大金额等于前 `i-1` 个房屋的结果 → `dp[i-1]`
- **偷**：不能偷第 `i-1` 个，只能从前 `i-2` 个的结果加上当前 → `dp[i-2] + nums[i-1]`

$$
dp[i] = \max(dp[i-1],\; dp[i-2] + nums[i-1])
$$

> 和爬楼梯的区别：爬楼梯是加法（方案数累加），这里是取 max（最优决策）。

### ③ 初始化

- `dp[0] = 0`（没有房屋可偷）
- `dp[1] = nums[0]`（只有一间，必偷）

### ④ 遍历顺序

`i` 从 2 到 `n` 正向。

### ⑤ 返回值

`dp[n]`。

## Python 代码

### 写法一：朴素 DP

```python
class Solution:
    def rob(self, nums: list[int]) -> int:
        n = len(nums)
        if n == 1:
            return nums[0]
        dp = [0] * (n + 1)
        dp[1] = nums[0]
        for i in range(2, n + 1):
            dp[i] = max(dp[i-1], dp[i-2] + nums[i-1])
        return dp[n]
```

### 写法二：滚动变量（O(1) 空间）

```python
class Solution:
    def rob(self, nums: list[int]) -> int:
        prev, curr = 0, 0          # dp[i-2], dp[i-1]
        for num in nums:
            prev, curr = curr, max(curr, prev + num)
        return curr
```

### 写法三：记忆化搜索

```python
from functools import cache

class Solution:
    def rob(self, nums: list[int]) -> int:
        n = len(nums)

        @cache
        def f(i: int) -> int:
            """前 i 个房屋能偷到的最大金额。"""
            if i <= 0:
                return 0
            if i == 1:
                return nums[0]
            return max(f(i-1), f(i-2) + nums[i-1])

        return f(n)
```

| 写法 | 时间 | 空间 |
|---|---|---|
| 朴素 DP | O(n) | O(n) |
| 滚动变量 | O(n) | O(1) |
| 记忆化搜索 | O(n) | O(n) |

## 易错点

1. **`dp[1] = nums[0]` 不是 `nums[1]`**：`dp[i]` 的定义是"前 i 个房屋"，i=1 对应 `nums[0]`。
2. **状态定义要严格**：是"前 i 个的最大值"还是"以第 i 个结尾的最大值"？这里用"前 i 个"，含义更自然。
3. **滚动赋值顺序**：`prev, curr = curr, max(curr, prev + num)` 必须同时赋值。
4. **边界 n=1**：朴素写法中如果 `n=1`，`range(2, 2)` 不进循环，返回 `dp[1]`，但前提是先做了 `n==1` 判断。

## 知识点补充

### 选/不选模型

这是线性 DP 中最常见的模型之一：每个元素有"选"或"不选"两种决策，带约束条件（这里是不能选相邻的）。

类似题目：
- **带约束的选/不选** → 打家劫舍系列
- **带权选/不选** → 背包问题

### 进阶：打家劫舍 II（213）

房屋排成**环形**（首尾相邻）。解法：分别算"不偷第一家"和"不偷最后一家"，取 max。

### 进阶：打家劫舍 III（337）

房屋排成**二叉树**。解法：树形 DP，每个节点返回 `(不偷该节点的最大值, 偷该节点的最大值)`。
