# 1143. 最长公共子序列

> 链接：https://leetcode.cn/problems/longest-common-subsequence/

## 题目描述

两个字符串 `text1` 和 `text2`，返回最长**公共子序列**的长度。子序列不要求连续。

### 示例

```
输入：text1 = "abcde", text2 = "ace"
输出：3
解释：最长公共子序列是 "ace"，长度为 3。
```

```
输入：text1 = "abc", text2 = "abc"
输出：3
```

```
输入：text1 = "abc", text2 = "def"
输出：0
```

约束：`1 <= text1.length, text2.length <= 1000`

---

## 思路推导（DP 五步法）

### ① 定义状态

`dp[i][j]` = `text1[0..i-1]` 和 `text2[0..j-1]` 的最长公共子序列长度。

> 注意：`i, j` 是**前缀长度**（1-indexed），不是索引。`dp[i][j]` 对应前 i 个和前 j 个字符。

### ② 写出转移方程

分两种情况：

**字符相同**（`text1[i-1] == text2[j-1]`）：

```math
dp[i][j] = dp[i-1][j-1] + 1
```

这两个字符配对，加上去掉它们后的 LCS。

**字符不同**（`text1[i-1] != text2[j-1]`）：

```math
dp[i][j] = \max(dp[i-1][j],\; dp[i][j-1])
```

要么去掉 `text1` 的当前字符，要么去掉 `text2` 的当前字符，取较大值。

### ③ 初始化

`dp[0][j] = 0`，`dp[i][0] = 0`（其中一个为空串时 LCS 为 0）。

### ④ 遍历顺序

`i` 从 1 到 `m`，`j` 从 1 到 `n`，按行遍历。

### ⑤ 返回值

`dp[m][n]`。

## Python 代码

### 写法一：二维 DP

```python
class Solution:
    def longestCommonSubsequence(self, text1: str, text2: str) -> int:
        m, n = len(text1), len(text2)
        dp = [[0] * (n + 1) for _ in range(m + 1)]

        for i in range(1, m + 1):
            for j in range(1, n + 1):
                if text1[i-1] == text2[j-1]:
                    dp[i][j] = dp[i-1][j-1] + 1
                else:
                    dp[i][j] = max(dp[i-1][j], dp[i][j-1])

        return dp[m][n]
```

### 写法二：一维 DP（空间优化）

只需要两行（上一行和当前行），甚至可以只用一行 + 一个临时变量。

```python
class Solution:
    def longestCommonSubsequence(self, text1: str, text2: str) -> int:
        m, n = len(text1), len(text2)
        # 确保 text2 是较短的串，节省空间
        if m < n:
            text1, text2 = text2, text1
            m, n = n, m

        prev = [0] * (n + 1)
        for i in range(1, m + 1):
            curr = [0] * (n + 1)
            for j in range(1, n + 1):
                if text1[i-1] == text2[j-1]:
                    curr[j] = prev[j-1] + 1
                else:
                    curr[j] = max(prev[j], curr[j-1])
            prev = curr
        return prev[n]
```

### 写法三：一行数组（极致压缩）

```python
class Solution:
    def longestCommonSubsequence(self, text1: str, text2: str) -> int:
        m, n = len(text1), len(text2)
        dp = [0] * (n + 1)
        for i in range(1, m + 1):
            prev = 0       # dp[i-1][j-1]
            for j in range(1, n + 1):
                temp = dp[j]   # 保存旧的 dp[i-1][j]
                if text1[i-1] == text2[j-1]:
                    dp[j] = prev + 1
                else:
                    dp[j] = max(dp[j], dp[j-1])
                prev = temp    # 更新对角线值
        return dp[n]
```

| 二维 DP | O(mn) | O(mn) |
| 两行 DP | O(mn) | O(n) |
| 一行 DP | O(mn) | O(n) |

## 易错点

1. **状态定义是"前缀长度"不是"索引"**：`dp[i][j]` 对应 `text1[0..i-1]` 和 `text2[0..j-1]`，所以字符比较用 `text1[i-1]`。
2. **字符不相等时是 `max(dp[i-1][j], dp[i][j-1])`**：不是 `dp[i-1][j-1]`（两个都去掉），而是只去掉一个取最大。
3. **一行压缩时保存 `prev`**：`dp[i-1][j-1]`（对角线值）会被 `dp[j]` 的新值覆盖，所以更新前要先保存。

## 知识点补充

### 双串 DP 通用模板

两个字符串/序列的 DP 问题几乎都用这个模式：

```
dp[i][j] = 最优解(text1[0..i-1], text2[0..j-1])

if text1[i-1] == text2[j-1]:
    dp[i][j] = f(dp[i-1][j-1])      # 匹配
else:
    dp[i][j] = g(dp[i-1][j], dp[i][j-1])  # 不匹配
```

| 题目 | 匹配 | 不匹配 |
|---|---|---|
| 1143 LCS | `dp[i-1][j-1] + 1` | `max(dp[i-1][j], dp[i][j-1])` |
| [[72-编辑距离]] | `dp[i-1][j-1]` | `1 + min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1])` |

### 如何还原具体的 LCS？

从 `dp[m][n]` 回溯：如果 `text1[i-1] == text2[j-1]`，该字符属于 LCS，走 `dp[i-1][j-1]`；否则走较大的方向。
