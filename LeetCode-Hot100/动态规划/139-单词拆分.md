# 139. 单词拆分

> 链接：https://leetcode.cn/problems/word-break/

## 题目描述

字符串 `s` 和字符串列表 `wordDict`，判断 `s` 是否可以被拆分为一个或多个字典中的单词。字典中的单词可以重复使用。

### 示例

```
输入：s = "leetcode", wordDict = ["leet", "code"]
输出：true
解释：s 可以拆分为 "leet" + "code"。
```

```
输入：s = "applepenapple", wordDict = ["apple", "pen"]
输出：true
解释：s = "apple" + "pen" + "apple"，"apple" 重复使用了。
```

```
输入：s = "catsandog", wordDict = ["cats", "dog", "sand", "and", "cat"]
输出：false
```

约束：`1 <= s.length <= 300`，`1 <= wordDict.length <= 1000`

---

## 思路推导（DP 五步法）

### ① 定义状态

`dp[j]` = `s[0:j]`（前 j 个字符）能否被拆分为字典中的单词（布尔值）。

### ② 写出转移方程

对每个位置 `j`，检查是否有某个单词 `w` 使得：

```math
dp[j] = \bigvee_{w \in wordDict,\; j \ge |w|} \big(dp[j - |w|] \;\wedge\; s[j-|w|:j] == w\big)
```

含义：如果 `s[0:j-|w|]` 可拆分 **且** `s[j-|w|:j]` 恰好是字典中的单词 `w`，则 `s[0:j]` 可拆分。

### ③ 初始化

- `dp[0] = True`（空字符串视为可拆分）
- `dp[j] = False`（`j > 0`）

### ④ 遍历顺序

外层 `j` 从 1 到 `n`，内层检查每个 `word`。

> 也可以外层 word 内层 j，但对本题效果类似。

### ⑤ 返回值

`dp[n]`。

## Python 代码

### 写法一：DP（外层位置，内层单词）

```python
class Solution:
    def wordBreak(self, s: str, wordDict: list[str]) -> bool:
        n = len(s)
        word_set = set(wordDict)
        dp = [False] * (n + 1)
        dp[0] = True

        for j in range(1, n + 1):
            for w in word_set:
                wlen = len(w)
                if j >= wlen and dp[j - wlen] and s[j - wlen:j] == w:
                    dp[j] = True
                    break     # 只要有一个能拼出就行
        return dp[n]
```

### 写法二：DP（优化内层遍历）

```python
class Solution:
    def wordBreak(self, s: str, wordDict: list[str]) -> bool:
        n = len(s)
        dp = [False] * (n + 1)
        dp[0] = True
        max_len = max(len(w) for w in wordDict) if wordDict else 0
        word_set = set(wordDict)

        for j in range(1, n + 1):
            # 只看可能的长度范围
            for l in range(1, min(j, max_len) + 1):
                if dp[j - l] and s[j - l:j] in word_set:
                    dp[j] = True
                    break
        return dp[n]
```

### 写法三：BFS

```python
from collections import deque

class Solution:
    def wordBreak(self, s: str, wordDict: list[str]) -> bool:
        word_set = set(wordDict)
        n = len(s)
        visited = set()
        queue = deque([0])

        while queue:
            start = queue.popleft()
            if start in visited:
                continue
            visited.add(start)
            for end in range(start + 1, n + 1):
                if s[start:end] in word_set:
                    if end == n:
                        return True
                    queue.append(end)
        return False
```

## 复杂度分析

| 写法 | 时间 | 空间 |
|---|---|---|
| DP 写法一 | O(n × m × L) | O(n) |
| DP 写法二 | O(n × L × L) | O(n) |
| BFS | O(n × L) | O(n) |

其中 m = wordDict 长度，L = 单词最大长度。

---

## 易错点

1. **`dp[0] = True`**：空字符串是基础情况，必须设为 True，否则递推永远无法启动。
2. **子串切片 `s[j-wlen:j]`**：Python 切片左闭右开，注意长度对齐。
3. **`break` 优化**：一旦 `dp[j]` 为 True 就跳出内层，不需要继续检查。
4. **`wordDict` 转 `set`**：列表查 `in` 是 O(n)，集合是 O(1)。
