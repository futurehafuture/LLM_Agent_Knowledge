# 62. 不同路径

> 链接：https://leetcode.cn/problems/unique-paths/

## 题目描述

`m × n` 网格，机器人位于左上角 `(0, 0)`，每次只能向右或向下移动一步，到达右下角 `(m-1, n-1)` 有多少条不同路径？

### 示例

```
输入：m = 3, n = 7
输出：28
```

```
输入：m = 3, n = 2
输出：3
解释：右→右→下 / 右→下→右 / 下→右→右
```

约束：`1 <= m, n <= 100`

---

## 思路推导（DP 五步法）

### ① 定义状态

`dp[i][j]` = 从 `(0,0)` 到 `(i,j)` 的不同路径数。

### ② 写出转移方程

到达 `(i,j)` 只能从上方 `(i-1,j)` 或左方 `(i,j-1)` 来：

```math
dp[i][j] = dp[i-1][j] + dp[i][j-1]
```

### ③ 初始化

第一行和第一列只有一条路径（一直向右或一直向下）：

- `dp[0][j] = 1` for all j
- `dp[i][0] = 1` for all i

### ④ 遍历顺序

`i` 从 0 到 `m-1`，`j` 从 0 到 `n-1`（任意顺序都行，只要保证 `dp[i-1][j]` 和 `dp[i][j-1]` 已算出）。

### ⑤ 返回值

`dp[m-1][n-1]`。

## Python 代码

### 写法一：二维 DP

```python
class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        dp = [[1] * n for _ in range(m)]
        for i in range(1, m):
            for j in range(1, n):
                dp[i][j] = dp[i-1][j] + dp[i][j-1]
        return dp[m-1][n-1]
```

### 写法二：一维 DP（空间优化）

`dp[j]` 只依赖上一行的 `dp[j]`（旧值）和本行的 `dp[j-1]`（新值），可以压缩成一维：

```python
class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        dp = [1] * n
        for _ in range(1, m):
            for j in range(1, n):
                dp[j] += dp[j-1]    # 等价于 dp[j] = dp[j] + dp[j-1]
        return dp[n-1]
```

### 写法三：组合数学

从 `(0,0)` 到 `(m-1,n-1)` 需要走 `m-1` 步下、`n-1` 步右，共 `m+n-2` 步。从中选 `m-1` 步向下：

```math
C(m+n-2,\; m-1) = \frac{(m+n-2)!}{(m-1)!(n-1)!}
```

```python
import math

class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        return math.comb(m + n - 2, m - 1)
```

## 复杂度分析

| 写法 | 时间 | 空间 |
|---|---|---|
| 二维 DP | O(mn) | O(mn) |
| 一维 DP | O(mn) | O(n) |
| 组合数学 | O(min(m,n)) | O(1) |

---

## 易错点

1. **初始化第一行第一列为 1**：只有一条路可以走到（全右/全下）。
2. **一维优化中 `j` 从 1 开始**：`dp[0]` 始终为 1（第一列），不需要更新。
3. **组合数可能溢出**：如果用其他语言，注意中间结果可能超出 int 范围。Python 没这个问题。

## 知识点补充

### 网格路径 DP 的通用模型

这类题的状态转移几乎都是：

```math
dp[i][j] = f(dp[i-1][j],\; dp[i][j-1])
```

不同的题只改 `f`：

| 题目 | f | 含义 |
|---|---|---|
| 62 不同路径 | `+` | 路径数累加 |
| [[64-最小路径和]] | `min + grid[i][j]` | 路径和取最小 |

### 障碍物版本（LeetCode 63）

如果有障碍物，初始化时遇到障碍设为 0，转移时跳过障碍格子。
