# 322. 零钱兑换

> 链接：https://leetcode.cn/problems/coin-change/

## 题目描述

整数数组 `coins` 表示不同面额的硬币，整数 `amount` 表示总金额。计算凑成总金额所需的**最少硬币数**。每种硬币可以重复使用。如果无法凑出，返回 `-1`。

### 示例

```
输入：coins = [1,2,5], amount = 11
输出：3
解释：11 = 5 + 5 + 1，用 3 枚硬币。
```

```
输入：coins = [2], amount = 3
输出：-1
```

```
输入：coins = [1], amount = 0
输出：0
```

约束：`1 <= coins.length <= 12`，`1 <= coins[i] <= 2^31-1`，`0 <= amount <= 10^4`

---

## 思路推导（DP 五步法）

### ① 定义状态

`dp[j]` = 凑出金额 `j` 所需的**最少硬币数**。

### ② 写出转移方程

对每个金额 `j`，尝试用每种硬币 `coin` 来凑：

$$
dp[j] = \min_{coin \in coins,\; j \ge coin} (dp[j],\; dp[j - coin] + 1)
$$

含义：不用这枚硬币（`dp[j]`）vs 用这枚硬币（`dp[j-coin] + 1`）。

### ③ 初始化

- `dp[0] = 0`（金额为 0 不需要硬币）
- `dp[j] = ∞`（`j > 0`，初始设为不可达）

### ④ 遍历顺序

外层硬币 `coin`，内层金额 `j` 从 `coin` 到 `amount` **正序**。

> 正序的原因：这是**完全背包**——硬币可以重复使用。正序遍历保证 `dp[j-coin]` 可能已经被同一轮更新过，对应"再用一次该硬币"。

### ⑤ 返回值

`dp[amount]` 如果不是 `∞`，否则返回 `-1`。

## Python 代码

### 写法一：完全背包（外层物品，内层容量正序）

```python
class Solution:
    def coinChange(self, coins: list[int], amount: int) -> int:
        dp = [float('inf')] * (amount + 1)
        dp[0] = 0
        for coin in coins:
            for j in range(coin, amount + 1):
                dp[j] = min(dp[j], dp[j - coin] + 1)
        return dp[amount] if dp[amount] != float('inf') else -1
```

### 写法二：外层容量，内层物品

```python
class Solution:
    def coinChange(self, coins: list[int], amount: int) -> int:
        dp = [float('inf')] * (amount + 1)
        dp[0] = 0
        for j in range(1, amount + 1):
            for coin in coins:
                if j >= coin:
                    dp[j] = min(dp[j], dp[j - coin] + 1)
        return dp[amount] if dp[amount] != float('inf') else -1
```

> 这两种写法在求 min 时等价（排列组合不影响最小值）。但求方案数时内外层顺序有区别。

### 写法三：BFS

```python
from collections import deque

class Solution:
    def coinChange(self, coins: list[int], amount: int) -> int:
        if amount == 0:
            return 0
        visited = [False] * (amount + 1)
        queue = deque([0])
        visited[0] = True
        steps = 0
        while queue:
            steps += 1
            for _ in range(len(queue)):
                cur = queue.popleft()
                for coin in coins:
                    nxt = cur + coin
                    if nxt == amount:
                        return steps
                    if nxt < amount and not visited[nxt]:
                        visited[nxt] = True
                        queue.append(nxt)
        return -1
```

## 复杂度分析

| 写法 | 时间 | 空间 |
|---|---|---|
| DP | O(amount × len(coins)) | O(amount) |
| BFS | O(amount × len(coins)) | O(amount) |

---

## 易错点

1. **初始化用 `inf` 不是 `-1`**：求最小值，初始应该为"不可能"即无穷大，这样 `min()` 自然会选有效值。
2. **内层正序**：完全背包物品可重复使用，必须正序。如果倒序就变成了 0/1 背包。
3. **返回值判断 `inf`**：最后要检查 `dp[amount]` 是否还是 `inf`。
4. **`amount = 0` 特判**：BFS 写法需要单独处理。
