# 152. 乘积最大子数组

> 链接：https://leetcode.cn/problems/maximum-product-subarray/

## 题目描述

整数数组 `nums`，找出乘积最大的非空连续子数组，返回该乘积。

### 示例

```
输入：nums = [2,3,-2,4]
输出：6
解释：子数组 [2,3] 的乘积最大，为 6。
```

```
输入：nums = [-2,0,-1]
输出：0
解释：结果不能是 2，因为 [-2,-1] 不是连续子数组。
```

约束：`1 <= nums.length <= 2*10^4`，`-10 <= nums[i] <= 10`

---

## 思路推导（DP 五步法）

### ① 定义状态

需要**同时维护最大值和最小值**（因为负负得正）：

- `maxDP[i]` = 以 `nums[i]` 结尾的子数组的最大乘积
- `minDP[i]` = 以 `nums[i]` 结尾的子数组的最小乘积

### ② 写出转移方程

对 `nums[i]`，有三种选择：

1. `nums[i]` 单独成为一个子数组
2. `nums[i]` 乘上以 `i-1` 结尾的最大乘积
3. `nums[i]` 乘上以 `i-1` 结尾的最小乘积（负 × 负 = 正）

$$
\begin{aligned}
maxDP[i] &= \max(nums[i],\; maxDP[i-1] \times nums[i],\; minDP[i-1] \times nums[i]) \\
minDP[i] &= \min(nums[i],\; maxDP[i-1] \times nums[i],\; minDP[i-1] \times nums[i])
\end{aligned}
$$

### ③ 初始化

`maxDP[0] = minDP[0] = nums[0]`

### ④ 遍历顺序

`i` 从 1 到 `n-1` 正向。

### ⑤ 返回值

`max(maxDP)`，即所有 `maxDP[i]` 中的最大值。

---

## Python 代码

### 写法一：朴素 DP（双数组）

```python
class Solution:
    def maxProduct(self, nums: list[int]) -> int:
        n = len(nums)
        maxDP = [0] * n
        minDP = [0] * n
        maxDP[0] = minDP[0] = nums[0]

        for i in range(1, n):
            candidates = (nums[i], maxDP[i-1] * nums[i], minDP[i-1] * nums[i])
            maxDP[i] = max(candidates)
            minDP[i] = min(candidates)

        return max(maxDP)
```

### 写法二：滚动变量（O(1) 空间）

```python
class Solution:
    def maxProduct(self, nums: list[int]) -> int:
        result = cur_max = cur_min = nums[0]
        for num in nums[1:]:
            candidates = (num, cur_max * num, cur_min * num)
            cur_max = max(candidates)
            cur_min = min(candidates)
            result = max(result, cur_max)
        return result
```

| 滚动变量 | O(n) | O(1) |

## 易错点

1. **必须同时维护最小值**：只维护最大值会漏掉"负 × 负 = 正"的情况。例如 `[-2, 3, -4]`，中间 `3 × (-2) = -6` 看起来很小，但 `(-6) × (-4) = 24` 才是答案。
2. **`candidates` 三取 max/min**：不要忘记 `nums[i]` 本身（单独一个元素的情况）。
3. **返回值不是 `maxDP[-1]`**：最大乘积子数组不一定以最后一个元素结尾。
4. **滚动更新要同时**：`cur_max` 和 `cur_min` 必须基于旧的值同时更新，不能先更新 `cur_max` 再算 `cur_min`。

## 知识点补充

### 为什么需要双状态？

这是 DP 中的经典技巧：当转移涉及**乘法**时，最小值可能在乘以负数后翻转为最大值。因此需要同时追踪极值的两个方向。

类似的双状态技巧：
- **乘积最×子数组** → 维护 max/min
- **股票买卖含冷冻期** → 维护持有/不持有

### 和 53 最大子数组和的区别

| 题目 | 操作 | 状态数 | 转移 |
|---|---|---|---|
| 53 最大子数组和 | 加法 | 1 | `max(nums[i], dp[i-1]+nums[i])` |
| 152 乘积最大子数组 | 乘法 | 2 | `max/min(nums[i], dp*nums[i])` |

加法不会翻转符号，所以只需要一个状态；乘法会，所以需要两个。
