# 74. 搜索二维矩阵

> 链接：https://leetcode.cn/problems/search-a-2d-matrix/

## 题目描述

在 `m × n` 的整数矩阵 `matrix` 中搜索目标值 `target`。

矩阵满足两个条件：
1. **每行**从左到右升序排列
2. **每行的第一个元素**大于上一行的最后一个元素

如果 `target` 在矩阵中，返回 `true`；否则返回 `false`。

### 示例

```
输入：matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,60]], target = 3
输出：true
```

```
输入：matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,60]], target = 13
输出：false
```

约束：`m, n >= 1`，`-10^4 <= matrix[i][j], target <= 10^4`

---

## 思路推导

### 核心洞察：矩阵可以"展平"

题目给出的两个条件合在一起意味着：**把矩阵按行拼接后，整个序列是严格升序的**。

```
[1, 3, 5, 7, 10, 11, 16, 20, 23, 30, 34, 60]
```

不需要真的构建这个一维数组，只需要做**索引映射**：

$$\text{idx} \quad \rightarrow \quad \text{row} = \text{idx} \,/\, n, \quad \text{col} = \text{idx} \,\%\, n$$

其中 `n` 是列数（不是行数）。然后标准二分即可。

---

## Python 代码

### 写法一：展平二分

```python
class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        m, n = len(matrix), len(matrix[0])
        l, r = 0, m * n - 1
        while l <= r:
            mid = l + (r - l) // 2
            row, col = divmod(mid, n)               # mid // n, mid % n
            val = matrix[row][col]
            if val == target:
                return True
            elif val < target:
                l = mid + 1
            else:
                r = mid - 1
        return False
```

### 写法二：两次二分（先定位行，再定位列）

```python
class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        m, n = len(matrix), len(matrix[0])

        # 第一次二分：定位目标行（找最后一个首元素 ≤ target 的行）
        l, r = 0, m - 1
        while l <= r:
            mid = l + (r - l) // 2
            if matrix[mid][0] <= target:
                l = mid + 1
            else:
                r = mid - 1
        row = r
        if row < 0:
            return False

        # 第二次二分：在目标行中二分查找
        l, r = 0, n - 1
        while l <= r:
            mid = l + (r - l) // 2
            if matrix[row][mid] == target:
                return True
            elif matrix[row][mid] < target:
                l = mid + 1
            else:
                r = mid - 1
        return False
```

## 复杂度分析

| 写法 | 时间 | 空间 |
|---|---|---|
| 展平二分 | O(log(mn)) | O(1) |
| 两次二分 | O(log m + log n) | O(1) |

`log(mn) = log m + log n`，两者渐进复杂度相同。

---

## 易错点

1. **行列映射时用 n（列数）而非 m**：`mid // n` 是行号，`mid % n` 是列号。用成 `m` 就错了。
2. **两次二分法中 row = r 而非 l**：`r` 停在最后一个满足 `matrix[r][0] <= target` 的行，`l = r + 1`。
3. **空矩阵边界**：`matrix` 或 `matrix[0]` 为空时需要处理（但题目约束 `m, n >= 1`）。

## 知识点补充

### 与 LC240 的区别

| 特征 | LC74 | LC240 |
|---|---|---|
| 矩阵性质 | 行内有序 + 行间有序（展平后整体有序） | 行内有序 + 列内有序（但展平后不一定有序） |
| 解法 | 展平二分 O(log mn) | 从右上/左下角遍历 O(m + n) 或二分 |
| 可以二分吗？ | 可以 | 每行分别二分 O(m log n) |

本题的条件更强，所以可以直接展平做标准二分。

### `divmod` 函数

```python
row, col = divmod(mid, n)
# 等价于
row = mid // n
col = mid % n
```
