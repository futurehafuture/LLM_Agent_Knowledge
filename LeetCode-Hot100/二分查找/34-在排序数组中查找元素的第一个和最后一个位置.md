# 34. 在排序数组中查找元素的第一个和最后一个位置

> 链接：https://leetcode.cn/problems/find-first-and-last-position-of-element-in-sorted-array/

## 题目描述

给定一个**非递减**排列的整数数组 `nums` 和一个目标值 `target`，找出 `target` 在数组中的**开始位置和结束位置**。

如果 `target` 不存在，返回 `[-1, -1]`。

要求时间复杂度 **O(log n)**。

### 示例

```
输入：nums = [5,7,7,8,8,10], target = 8
输出：[3,4]
```

```
输入：nums = [5,7,7,8,8,10], target = 6
输出：[-1,-1]
```

```
输入：nums = [], target = 0
输出：[-1,-1]
```

约束：`0 <= nums.length <= 10^5`

---
```python
# bilib
from typing import List

class Solution:
    def searchRange(self, nums: List[int], target: int) -> List[int]:
        n = len(nums)

        if not n:
            return [-1, -1]

        i, j = 0, n - 1

        while i <= j:
            mid = i + (j - i) // 2

            if target > nums[mid]:
                i = mid + 1
            elif target < nums[mid]:
                j = mid - 1
            else:
                x, y = mid, mid

                while x - 1 >= 0 and nums[x - 1] == target:
                    x = x - 1

                while y + 1 <= n - 1 and nums[y + 1] == target:
                    y = y + 1

                return [x, y]

        return [-1, -1]
```



## 思路推导

### 问题拆解

要找两个位置：
- **左边界**：第一个 **≥ target** 的位置（`lower_bound(target)`）
- **右边界**：最后一个 **≤ target** 的位置（`lower_bound(target+1) - 1`）

或者两次二分，分别找左边界和右边界。

### 左边界二分

当 `nums[mid] >= target` 时收缩 `r`；否则收缩 `l`。循环结束 `l` 指向第一个 ≥ target 的位置。

### 右边界二分

当 `nums[mid] <= target` 时收缩 `l`；否则收缩 `r`。循环结束 `r` 指向最后一个 ≤ target 的位置。

### 验证存在性

找到左边界后，检查 `l` 是否越界、`nums[l]` 是否等于 `target`。

## Python 代码

### 写法一：两次 lower_bound（最简洁）

```python
class Solution:
    def searchRange(self, nums: List[int], target: int) -> List[int]:
        def lower_bound(x):
            l, r = 0, len(nums) - 1
            while l <= r:
                mid = l + (r - l) // 2
                if nums[mid] >= x:
                    r = mid - 1
                else:
                    l = mid + 1
            return l

        left = lower_bound(target)
        if left == len(nums) or nums[left] != target:
            return [-1, -1]
        right = lower_bound(target + 1) - 1
        return [left, right]
```

### ## 复杂度分析

| 写法             | 时间       | 空间   |
| -------------- | -------- | ---- |
| 两次 lower_bound | O(log n) | O(1) |

## 易错点

1. **忘记验证存在性**：左边界二分只保证 `l` 停在 ≥ target 的位置，但不保证 `nums[l] == target`。必须检查。
2. **`lower_bound(target+1) - 1` 的边界**：当 `target` 是最大值时，`lower_bound(target+1)` 返回 `len(nums)`，`-1` 后是 `len(nums)-1`，恰好对。
3. **空数组**：left 检查 `left == len(nums)` 覆盖了空数组和 target 大于所有元素的情况。
4. **右边界返回 r 而非 l**：在"找最后一个 ≤ target"的二分中，循环结束时 `r < l`，`r` 才是合法位置。

## 知识点补充

### lower_bound 的巧用

找右边界不需要另写一个函数，`lower_bound(target+1) - 1` 等价于找最后一个 `target` 的位置。

原理：第一个 ≥ target+1 的位置的前一个位置，要么是 target（如果有），要么小于 target（如果 target 不存在）。

### 边界二分模板总结

| 目标 | 条件 | l 更新 | r 更新 | 返回值 |
|---|---|---|---|---|
| 第一个 ≥ x | `nums[mid] >= x` | `mid + 1` | `mid - 1` | `l` |
| 第一个 > x | `nums[mid] > x` | `mid + 1` | `mid - 1` | `l` |
| 最后一个 < x | `nums[mid] < x` | `mid + 1` | `mid - 1` | `r` |
| 最后一个 ≤ x | `nums[mid] <= x` | `mid + 1` | `mid - 1` | `r` |

规律：**更新 l 的写在 if 里 → 返回 r；更新 r 的写在 if 里 → 返回 l**。
