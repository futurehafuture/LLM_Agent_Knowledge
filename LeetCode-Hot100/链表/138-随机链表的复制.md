# 138. 随机链表的复制

> 链接：https://leetcode.cn/problems/copy-list-with-random-pointer/

## 题目描述

给你一个长度为 `n` 的链表，每个节点包含一个额外的**随机指针** `random`，该指针可以指向链表中的任何节点或空节点。

构造这个链表的**深拷贝**。深拷贝应该正好由 `n` 个全新节点组成，其中每个新节点的值与对应的原节点相同，`next` 和 `random` 指针也都指向复制链表中的新节点，并且这些指针应当表示与原始链表中相同的链表状态。

返回复制链表的头节点。

```python
class Node:
    def __init__(self, x: int, next=None, random=None):
        self.val = int(x)
        self.next = next
        self.random = random
```

### 示例

```
输入：head = [[7,null],[13,0],[11,4],[10,2],[1,0]]
输出：[[7,null],[13,0],[11,4],[10,2],[1,0]]
（每对表示 [val, random 指向的原链表节点索引]）
```

---

## 思路推导

### 方法一：哈希表（两次遍历）

**核心问题**：复制 `random` 指针时，目标节点可能还没创建。

**解法**：先创建所有新节点，用哈希表 `old → new` 建立映射，再第二次遍历补上 `next` 和 `random`。

### 方法二：原地穿插（O(1) 空间）

**三步走**：

**Step 1**：在每个原节点后面插入其克隆节点：

```
原：1 → 2 → 3 → None
变：1 → 1' → 2 → 2' → 3 → 3' → None
```

**Step 2**：设置克隆节点的 `random`：
- `node.next.random = node.random.next`（`node.random` 的克隆就在它后面）

**Step 3**：拆分两条链表，恢复原链表并提取克隆链表。

## Python 代码

### 写法一：哈希表（推荐，简洁）

```python
class Solution:
    def copyRandomList(self, head: Optional[Node]) -> Optional[Node]:
        if not head:
            return None
        # 建立旧节点 → 新节点 的映射
        old_to_new = {}
        curr = head
        while curr:
            old_to_new[curr] = Node(curr.val)
            curr = curr.next
        # 补充 next 和 random 指针
        curr = head
        while curr:
            if curr.next:
                old_to_new[curr].next = old_to_new[curr.next]
            if curr.random:
                old_to_new[curr].random = old_to_new[curr.random]
            curr = curr.next
        return old_to_new[head]
```

### 写法二：原地穿插（O(1) 空间）

```python
class Solution:
    def copyRandomList(self, head: Optional[Node]) -> Optional[Node]:
        if not head:
            return None
        # Step 1：在每个节点后插入克隆节点
        curr = head
        while curr:
            clone = Node(curr.val, curr.next)
            curr.next = clone
            curr = clone.next
        # Step 2：设置克隆节点的 random
        curr = head
        while curr:
            if curr.random:
                curr.next.random = curr.random.next
            curr = curr.next.next
        # Step 3：拆分链表
        dummy = Node(0)
        clone_curr = dummy
        curr = head
        while curr:
            clone_curr.next = curr.next
            curr.next = curr.next.next
            clone_curr = clone_curr.next
            curr = curr.next
        return dummy.next
```

## 复杂度分析

| 写法 | 时间 | 空间 |
|---|---|---|
| 哈希表 | O(n) | O(n) |
| 原地穿插 | O(n) | O(1) |

---

## 易错点

1. **哈希表写法的 `None` 处理**：`curr.random` 可能是 None，不能无条件 `old_to_new[curr.random]`，需要 `if curr.random` 判断。
2. **原地穿插 Step 2 中 `curr.random.next` 的前提**：只有 `curr.random` 不为 None 时才能取 `curr.random.next`，`curr.next.random` 默认是 None，不需要额外处理 random 为 None 的情况。
3. **Step 3 拆分时恢复原链表**：`curr.next = curr.next.next` 是恢复原链表，不能少；如果漏写，原链表被破坏（面试官通常会检查原链表是否完整）。

## 知识点补充

### 递归 + 哈希表（记忆化）

也可以用 DFS 递归复制，哈希表防止重复创建：

```python
class Solution:
    def copyRandomList(self, head: Optional[Node]) -> Optional[Node]:
        memo = {}
        def dfs(node):
            if not node: return None
            if node in memo: return memo[node]
            clone = Node(node.val)
            memo[node] = clone
            clone.next = dfs(node.next)
            clone.random = dfs(node.random)
            return clone
        return dfs(head)
```

三种写法（迭代哈希 / 原地穿插 / 递归哈希）都要能写，面试常追问空间优化。
