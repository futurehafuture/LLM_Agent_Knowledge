# 21. 合并两个有序链表

> 链接：https://leetcode.cn/problems/merge-two-sorted-lists/

## 题目描述

将两个升序链表合并为一个新的**升序**链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

### 示例

```
输入：list1 = [1,2,4]，list2 = [1,3,4]
输出：[1,1,2,3,4,4]
```

```
输入：list1 = []，list2 = []
输出：[]
```

```
输入：list1 = []，list2 = [0]
输出：[0]
```

约束：两个链表的节点数在 `[0, 50]` 范围内，`-100 <= Node.val <= 100`。

---

## 思路推导

### 迭代法：虚拟头节点 + 双指针

引入 `dummy` 节点作为合并结果的虚拟头，避免对第一个节点进行特判。

维护 `tail` 指针指向当前已合并链表的末尾，每次从 `l1` 和 `l2` 中取较小的节点接上去，推进对应指针。

两个链表都非空时一直比较；某一方先耗尽，把另一方剩余部分直接接上（因为已经有序，直接接不需要再遍历）。

### 递归法

合并等于：
- 如果 `l1.val <= l2.val`：`l1.next = merge(l1.next, l2)`，返回 `l1`
- 否则：`l2.next = merge(l1, l2.next)`，返回 `l2`

---

## Python 代码

### 写法一：迭代（推荐）

```python
class Solution:
    def mergeTwoLists(self, list1: Optional[ListNode], list2: Optional[ListNode]) -> Optional[ListNode]:
        dummy = ListNode(0)
        tail = dummy
        while list1 and list2:
            if list1.val <= list2.val:
                tail.next = list1
                list1 = list1.next
            else:
                tail.next = list2
                list2 = list2.next
            tail = tail.next
        tail.next = list1 or list2   # 接上剩余部分
        return dummy.next
```

### 写法二：递归

```python
class Solution:
    def mergeTwoLists(self, list1: Optional[ListNode], list2: Optional[ListNode]) -> Optional[ListNode]:
        if not list1:
            return list2
        if not list2:
            return list1
        if list1.val <= list2.val:
            list1.next = self.mergeTwoLists(list1.next, list2)
            return list1
        else:
            list2.next = self.mergeTwoLists(list1, list2.next)
            return list2
```

## 复杂度分析

| 写法 | 时间 | 空间 |
|---|---|---|
| 迭代 | O(m+n) | O(1) |
| 递归 | O(m+n) | O(m+n)（调用栈） |

## 易错点

1. **`tail.next = list1 or list2`**：Python 的 `or` 返回第一个真值，两者都是 None 时返回 None，任意一方有剩余时返回那一方的当前节点。简洁且正确。
2. **迭代中不要忘记 `tail = tail.next`**：接上节点后 tail 要推进，否则每次都把新节点接到同一个位置。
3. **不要新建节点**：直接复用原链表节点，不需要 `ListNode(list1.val)` 这样新建。

## 知识点补充

### 虚拟头节点的作用

没有 dummy 时，第一个节点需要特判：

```python
# 没有 dummy，需要单独确定头节点
if list1.val <= list2.val:
    head = list1
    list1 = list1.next
else:
    head = list2
    list2 = list2.next
tail = head
# 后续主循环...
```

加了 `dummy` 之后，头节点和普通节点的处理逻辑完全统一，代码大幅简化。这是链表题的标准技巧。
