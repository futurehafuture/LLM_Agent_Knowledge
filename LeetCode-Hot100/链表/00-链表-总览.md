# 链表总览

> [!tip] 给新手的一句话
> 链表题的核心是**指针操作**：在纸上画出节点与箭头，再翻译成代码。90% 的链表题靠三种技巧解决：**虚拟头节点、快慢指针、递归/迭代反转**。

---

## 1. 链表基础（Python 定义）

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next
```

链表与数组的核心区别：

| 操作 | 数组 | 链表 |
|---|---|---|
| 随机访问 | O(1) | O(n) |
| 头部插入/删除 | O(n) | O(1) |
| 尾部插入/删除 | O(1)（摊销） | O(n) 无尾指针 |
| 中间插入/删除 | O(n) | O(1) 已有节点引用 |

## 2. 三大核心技巧

### 2.1 虚拟头节点（Dummy Node）

**适用**：需要修改头节点（删除头、合并、插入到头部前）。

```python
dummy = ListNode(0)
dummy.next = head
# ... 操作 ...
return dummy.next
```

> 加了 dummy 之后，所有节点的处理逻辑统一，不再需要单独判断头节点。

代表题：**21 合并两个有序链表、19 删除倒数第 N 个节点、24 两两交换**

---

### 2.2 快慢指针（Floyd's Tortoise and Hare）

**适用**：找中点、检测环、找入环点、找倒数第 k 个节点。

```python
slow = fast = head
while fast and fast.next:
    slow = slow.next
    fast = fast.next.next
# 循环结束时 slow 在中点
```

**速度比 1:2 的关键结论**：
- 奇数长度：slow 停在正中间
- 偶数长度：slow 停在中间偏左（左中位数）
- 有环时：fast 与 slow 必然相遇

代表题：**141 环形链表、142 环形链表 II、234 回文链表、876 链表的中间节点**

---

### 2.3 双指针（间距固定）

**适用**：找倒数第 k 个节点、找两链表的交点。

```python
# 找倒数第 k 个：先让 fast 走 k 步
fast = head
for _ in range(k):
    fast = fast.next
slow = head
while fast:
    slow = slow.next
    fast = fast.next
# slow 指向倒数第 k 个
```

代表题：**19 删除倒数第 N 个节点、160 相交链表**

## 3. 反转链表（三种写法）

所有反转类题目的基础：

### 迭代（最常考）

```python
def reverseList(head):
    prev, curr = None, head
    while curr:
        nxt = curr.next
        curr.next = prev
        prev = curr
        curr = nxt
    return prev
```

### 递归

```python
def reverseList(head):
    if not head or not head.next:
        return head
    new_head = reverseList(head.next)
    head.next.next = head
    head.next = None
    return new_head
```

### 反转区间 [left, right]（25 题的基础）

```python
def reverseBetween(head, left, right):
    dummy = ListNode(0, head)
    pre = dummy
    for _ in range(left - 1):
        pre = pre.next
    cur = pre.next
    for _ in range(right - left):
        nxt = cur.next
        cur.next = nxt.next
        nxt.next = pre.next
        pre.next = nxt
    return dummy.next
```

## 5. 常见易错点

1. **忘记 `nxt = curr.next`**：在修改 `curr.next` 之前，先保存后续节点。
2. **虚拟头节点忘记 `dummy.next = head`**：只 `ListNode(0)` 没有连起来，后续全错。
3. **快慢指针终止条件**：检测环用 `while fast and fast.next`；找中点奇偶行为不同，要在纸上验证。
4. **递归反转的两行顺序**：
   ```python
   head.next.next = head  # 先让 head.next 指回来
   head.next = None       # 再断开 head 的原指针
   # 顺序反了 → 形成新的环
   ```
5. **进位处理**：两数相加循环退出后要检查 `if carry`，别丢最高位。
6. **环形链表 II 的数学**：快慢指针相遇后，让一个指针回到 head，再同速走，相遇点才是入环口。
