# 24. 两两交换链表中的节点

> 链接：https://leetcode.cn/problems/swap-nodes-in-pairs/

## 题目描述

给你一个链表，两两交换其中相邻的节点，并返回交换后链表的头节点。你必须在不修改节点内部的值的情况下完成本题（即，只能进行节点交换）。

### 示例

```
输入：head = [1,2,3,4]
输出：[2,1,4,3]
```

```
输入：head = []
输出：[]
```

```
输入：head = [1]
输出：[1]
```

约束：节点数在 `[0, 100]` 范围内，`0 <= Node.val <= 100`。

---

## 思路推导

### 迭代法：虚拟头节点

每次处理 `pre → node1 → node2 → ...`，把 node1 和 node2 交换，变成 `pre → node2 → node1 → ...`。

交换三步：
1. `pre.next = node2`
2. `node1.next = node2.next`（先保存 node2 的后继）
3. `node2.next = node1`

然后 `pre = node1`（node1 现在在 node2 后面），继续处理后面的两个节点。

```
初始：dummy → 1 → 2 → 3 → 4 → None
       pre    n1  n2

交换后：dummy → 2 → 1 → 3 → 4 → None
                    pre  n1  n2（下一轮）
```

### 递归法

交换前两个节点，后续让递归处理：
```
swap([1,2,3,4]) = 2 → 1 → swap([3,4])
swap([3,4])     = 4 → 3 → swap([]) = None
```

## Python 代码

### 写法一：迭代（推荐）

```python
class Solution:
    def swapPairs(self, head: Optional[ListNode]) -> Optional[ListNode]:
        dummy = ListNode(0, head)
        pre = dummy
        while pre.next and pre.next.next:
            node1 = pre.next
            node2 = pre.next.next
            # 交换
            pre.next = node2
            node1.next = node2.next
            node2.next = node1
            # pre 推进到这对的末尾（node1 现在是后一个）
            pre = node1
        return dummy.next
```

### 写法二：递归

```python
class Solution:
    def swapPairs(self, head: Optional[ListNode]) -> Optional[ListNode]:
        if not head or not head.next:
            return head
        node1, node2 = head, head.next
        node1.next = self.swapPairs(node2.next)
        node2.next = node1
        return node2
```

## 复杂度分析

| 写法 | 时间 | 空间 |
|---|---|---|
| 迭代 | O(n) | O(1) |
| 递归 | O(n) | O(n)（调用栈） |

## 易错点

1. **迭代中三步赋值顺序**：必须先 `node1.next = node2.next` 保存 node2 的后继，再 `node2.next = node1`。如果先改 `node2.next`，`node2.next` 原有的后续链表就丢了。
2. **pre 推进到 `node1`（交换后的末尾节点）**：交换后 node1 在后面，下一轮要从 node1 后面取两个节点。
3. **循环条件 `pre.next and pre.next.next`**：两个节点都存在才交换；只剩一个节点就保持原样。

## 知识点补充

### 链表节点交换的通用方法

本题是 [[25-K个一组翻转链表]] 的特殊情况（K=2）。K=2 时只需交换两个节点，K>2 时需要反转整个子段。理解本题的三步赋值思路后，K 组翻转只是把"交换两个"换成"反转 K 个"。

### 递归写法的直觉

```
swapPairs(1→2→3→4)
= 2 → 1 → swapPairs(3→4)
= 2 → 1 → 4 → 3
```

递归写法更简洁，但要注意 `node1.next = swapPairs(node2.next)` 必须在 `node2.next = node1` 之前，否则 node2.next 已经指向 node1，递归调用的参数就错了。
