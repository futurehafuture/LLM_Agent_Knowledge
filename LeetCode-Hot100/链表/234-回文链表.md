# 234. 回文链表

> 链接：https://leetcode.cn/problems/palindrome-linked-list/

## 题目描述

给你一个单链表的头节点 `head`，请你判断该链表是否为回文链表。如果是，返回 `true`；否则，返回 `false`。

**进阶**：用 O(n) 时间复杂度和 O(1) 空间复杂度解决？

### 示例

```
输入：head = [1,2,2,1]
输出：true
```

```
输入：head = [1,2]
输出：false
```

约束：节点数在 `[1, 10^5]` 范围内，`0 <= Node.val <= 9`。

---

## 思路推导

### 最优方案：快慢指针 + 反转后半段

三步走：

**Step 1：快慢指针找中点**

```
slow 走 1 步，fast 走 2 步
1 → 2 → 2 → 1 → None
                fast先到None，slow停在中间偏左位置
```

- 偶数长链表（如 [1,2,2,1]）：slow 停在左半段最后节点（第 2 个 2 的前一个，即第 1 个 2）。
- 奇数长链表（如 [1,2,3,2,1]）：slow 停在正中间（3）。

**Step 2：反转后半段**

从 `slow.next` 开始反转。

**Step 3：双指针比较**

一个从 head 出发，一个从反转后的后半段头出发，逐步比较。

### 奇偶处理

奇数长链表反转后半段不包含中间元素，比较时右指针先到 `None` 即停止，中间元素无需参与比较，天然正确。

---

## Python 代码

### 写法一：O(1) 空间（快慢指针 + 反转）

```python
class Solution:
    def isPalindrome(self, head: Optional[ListNode]) -> bool:
        # Step 1：快慢指针找中点
        slow = fast = head
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next

        # Step 2：反转后半段
        prev, curr = None, slow.next
        while curr:
            nxt = curr.next
            curr.next = prev
            prev = curr
            curr = nxt
        second_half_head = prev

        # Step 3：双指针比较
        p1, p2 = head, second_half_head
        while p2:
            if p1.val != p2.val:
                return False
            p1 = p1.next
            p2 = p2.next
        return True
```

### 写法二：转数组（简单但 O(n) 空间）

```python
class Solution:
    def isPalindrome(self, head: Optional[ListNode]) -> bool:
        vals = []
        curr = head
        while curr:
            vals.append(curr.val)
            curr = curr.next
        return vals == vals[::-1]
```

## 复杂度分析

| 写法 | 时间 | 空间 |
|---|---|---|
| 快慢指针 + 反转 | O(n) | O(1) |
| 转数组 | O(n) | O(n) |

---

## 易错点

1. **反转的起点是 `slow.next` 而不是 `slow`**：slow 是左半段的最后节点，后半段从 `slow.next` 开始。
2. **奇数长度无需特判**：比较时用 `while p2`，后半段比前半段短（奇数时少一个中间节点），p2 先为 None 自然停止。
3. **比较时 `while p2` 而不是 `while p1 and p2`**：即使奇数长度 p1 还有剩余元素，p2 已结束即代表对称部分已全部比较完毕。

---

## 知识点补充

### 快慢指针找中点的奇偶行为

| 链表长度 | slow 停在 | 后半段起点 |
|---|---|---|
| 偶数（4 个：1,2,2,1） | 第 2 个节点（值 2） | slow.next（第 3 个，值 2） |
| 奇数（5 个：1,2,3,2,1） | 第 3 个节点（值 3，中间） | slow.next（第 4 个，值 2） |
