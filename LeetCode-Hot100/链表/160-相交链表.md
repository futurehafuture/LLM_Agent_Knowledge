# 160. 相交链表

> 链接：https://leetcode.cn/problems/intersection-of-two-linked-lists/

## 题目描述

给你两个单链表的头节点 `headA` 和 `headB`，请你找出并返回两个单链表相交的起始节点。如果两个链表不存在相交节点，返回 `null`。

> 相交：两个链表从某个节点开始共享同一条链（节点引用相同，不只是值相同）。

### 示例

```
A:       a1 → a2 ↘
                   c1 → c2 → c3
B: b1 → b2 → b3 ↗
输出：c1
```

```
A: 2 → 6 → 4
B: 1 → 5
输出：null
```

---

## 思路推导

### 方法一：哈希表

遍历链表 A，把所有节点存入 set；再遍历链表 B，第一个出现在 set 中的节点就是交点。

### 方法二：双指针消除长度差（O(1) 空间，推荐）

设链表 A 的非公共部分长 `a`，链表 B 的非公共部分长 `b`，公共部分长 `c`。

- 指针 `pA` 从 A 出发走 `a + c` 步到达 A 末尾，切换到 B 的头继续走 `b` 步到达交点，共走 `a + c + b` 步。
- 指针 `pB` 从 B 出发走 `b + c` 步到达 B 末尾，切换到 A 的头继续走 `a` 步到达交点，共走 `b + c + a` 步。

两者走的总步数相同，必然同时到达交点。

若无交点：两个指针会同时走到 `None`（`None is None` 为 `True`），循环结束返回 `None`。

```
pA: a1 → a2 → c1 → c2 → c3 → b1 → b2 → b3 → [c1]
pB: b1 → b2 → b3 → c1 → c2 → c3 → a1 → a2 → [c1]
                                               ↑ 相遇
```

## Python 代码

### 写法一：哈希表

```python
class Solution:
    def getIntersectionNode(self, headA: ListNode, headB: ListNode) -> Optional[ListNode]:
        seen = set()
        curr = headA
        while curr:
            seen.add(curr)
            curr = curr.next
        curr = headB
        while curr:
            if curr in seen:
                return curr
            curr = curr.next
        return None
```

### 写法二：双指针（O(1) 空间）

```python
class Solution:
    def getIntersectionNode(self, headA: ListNode, headB: ListNode) -> Optional[ListNode]:
        pA, pB = headA, headB
        while pA is not pB:
            pA = pA.next if pA else headB
            pB = pB.next if pB else headA
        return pA
```

## 复杂度分析

| 写法 | 时间 | 空间 |
|---|---|---|
| 哈希表 | O(m+n) | O(m) |
| 双指针 | O(m+n) | O(1) |

---

## 易错点

1. **切换时机**：`pA = pA.next if pA else headB`，是 `pA` 为 `None` 时切换到另一个头，而不是走完后第二步才切。如果写成 `if pA.next is None: pA = headB` 会多走一步，偏移一位。
2. **无交点时**：两个指针都从 `None` 切换为对方的头，最终同时走到 `None`，`None is None` 退出循环返回 `None`，无需特判。
3. **用 `is` 不用 `==`**：比较的是节点引用（同一对象），不是值。

## 知识点补充

### 消除长度差的本质

把两个链表"首尾相接"后，等价于两者都走了 `a + b + c` 步，自然对齐到同一个节点上。

这是一种非常优雅的"等价变换"思路：与其算出长度差然后让长链表先走差值步，不如让两指针互换起点，等效抵消差值。
