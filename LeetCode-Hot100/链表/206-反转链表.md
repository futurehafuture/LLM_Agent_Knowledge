# 206. 反转链表

> 链接：https://leetcode.cn/problems/reverse-linked-list/

## 题目描述

给你单链表的头节点 `head`，请你反转链表，并返回反转后的链表。

### 示例

```
输入：head = [1,2,3,4,5]
输出：[5,4,3,2,1]
```

```
输入：head = [1,2]
输出：[2,1]
```

```
输入：head = []
输出：[]
```

约束：链表节点数在 `[0, 5000]` 范围内，`-5000 <= Node.val <= 5000`。

---

## 思路推导

### 迭代法

维护 `prev`（前驱）和 `curr`（当前）两个指针，每步让 `curr.next` 指向 `prev`，然后整体向右滑动。

```
初始：None  →  1  →  2  →  3  →  4  →  5  →  None
      prev    curr

Step1：     None  ←  1     2  →  3  ...
            prev    curr  next（提前保存）

最终：None  ←  1  ←  2  ←  3  ←  4  ←  5
                                        prev（新头）
```

关键点：**必须在修改 `curr.next` 前先保存 `curr.next`**，否则链表断开就找不到下一个节点了。

### 递归法

把"反转从 `head.next` 开始的子链表"交给递归，拿到新头 `new_head` 后，只需处理 `head` 的回指：

```
反转 [1, 2, 3, 4, 5]
→ 递归反转 [2, 3, 4, 5]，得到 5→4→3→2，new_head = 5
→ 此时 1.next 仍是 2，让 2.next = 1，1.next = None
→ 返回 new_head = 5
```

## Python 代码

### 写法一：迭代（推荐）

```python
class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        prev, curr = None, head
        while curr:
            nxt = curr.next      # 保存下一个节点
            curr.next = prev     # 反转指针
            prev = curr          # prev 前进
            curr = nxt           # curr 前进
        return prev
```

### 写法二：递归

```python
class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        if not head or not head.next:
            return head
        new_head = self.reverseList(head.next)
        head.next.next = head   # head.next 回指 head
        head.next = None        # 断开 head 的原指针，避免成环
        return new_head
```

## 复杂度分析

| 写法 | 时间 | 空间 |
|---|---|---|
| 迭代 | O(n) | O(1) |
| 递归 | O(n) | O(n)（调用栈） |

## 易错点

1. **迭代法忘记 `nxt = curr.next`**：先改 `curr.next = prev` 就找不到后续节点，链表断了。
2. **递归法两行顺序不能颠倒**：
   ```python
   head.next = None        # ❌ 先断开 → head.next.next = head 操作的是 None
   head.next.next = head
   ```
   必须先 `head.next.next = head`，再 `head.next = None`。
3. **返回值是 `prev` 不是 `head`**：迭代结束时 `curr = None`，`prev` 才是新头。

## 知识点补充

### 反转区间 [left, right]

本题是 [[92-反转链表II]] 的基础（92 不在 Hot 100 但面试常考）。

反转区间技巧：头插法（O(1) 空间）：

```python
# 找到 left 的前驱 pre，curr = pre.next
for _ in range(right - left):
    nxt = curr.next
    curr.next = nxt.next
    nxt.next = pre.next
    pre.next = nxt
```
