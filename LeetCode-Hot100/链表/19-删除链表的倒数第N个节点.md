# 19. 删除链表的倒数第 N 个节点

> 链接：https://leetcode.cn/problems/remove-nth-node-from-end-of-list/

## 题目描述

给你一个链表，删除链表的倒数第 `n` 个节点，并且返回链表的头节点。

**进阶**：一趟扫描实现？

### 示例

```
输入：head = [1,2,3,4,5]，n = 2
输出：[1,2,3,5]
```

```
输入：head = [1]，n = 1
输出：[]
```

```
输入：head = [1,2]，n = 1
输出：[1]
```

约束：链表节点数为 `sz`，`1 <= sz <= 30`，`0 <= Node.val <= 100`，`1 <= n <= sz`。

---

## 思路推导

### 双指针（一趟扫描）

要删除倒数第 n 个节点，需要找到它的**前驱节点**（倒数第 n+1 个）。

**技巧**：让快指针先走 `n+1` 步，然后快慢指针同步走，当快指针到 `None` 时，慢指针正好在倒数第 n+1 个节点上。

```
删除倒数第 2 个（n=2），需要找到倒数第 3 个（前驱）：

dummy → 1 → 2 → 3 → 4 → 5 → None
fast 先走 n+1=3 步：
fast = 3（即 dummy 后第 3 个，值为 3）
slow = dummy

同步走直到 fast = None：
fast: 3 → 4 → 5 → None（走了 3 步）
slow: dummy → 1 → 2 → 3（也走了 3 步，停在倒数第 3 个）

slow.next = slow.next.next → 跳过节点 4
```

**为什么要用 dummy**：如果要删除的是头节点（链表长度 = n），slow 需要停在虚拟的"头节点前驱"，dummy 正好承担这个角色。

## Python 代码

### 双指针（推荐，一趟扫描）

```python
class Solution:
    def removeNthFromEnd(self, head: Optional[ListNode], n: int) -> Optional[ListNode]:
        dummy = ListNode(0, head)
        fast = slow = dummy
        # fast 先走 n+1 步
        for _ in range(n + 1):
            fast = fast.next
        # 同步走直到 fast 为 None
        while fast:
            fast = fast.next
            slow = slow.next
        # slow 现在是待删节点的前驱
        slow.next = slow.next.next
        return dummy.next
```

### 两趟扫描（先求长度）

```python
class Solution:
    def removeNthFromEnd(self, head: Optional[ListNode], n: int) -> Optional[ListNode]:
        dummy = ListNode(0, head)
        # 第一趟：求链表长度
        length = 0
        curr = head
        while curr:
            length += 1
            curr = curr.next
        # 第二趟：走到倒数第 n+1 个
        curr = dummy
        for _ in range(length - n):
            curr = curr.next
        curr.next = curr.next.next
        return dummy.next
```

| 写法 | 时间 | 空间 | 遍历次数 |
|---|---|---|---|
| 双指针 | O(n) | O(1) | 1 趟 |
| 两趟扫描 | O(n) | O(1) | 2 趟 |

---

## 易错点

1. **fast 先走 `n+1` 步，不是 `n` 步**：走 n 步 slow 会停在待删节点本身，无法执行删除（需要前驱）；走 n+1 步 slow 才停在前驱。
2. **初始 `fast = slow = dummy`（从 dummy 出发）**：如果从 head 出发，当 n 等于链表长度时（删除头节点），fast 走完 n+1 步会越界（None 再走会报错）。
3. **用 `dummy.next = head` 初始化**：`ListNode(0, head)` 等价于 `dummy = ListNode(0); dummy.next = head`，二选一即可。

---

## 知识点补充

### 双指针"间距"思维

把 fast 先走 k 步理解为：fast 和 slow 之间保持 k 个节点的间距。当 fast 到终点时，slow 距终点正好 k 步。

| 目标 | fast 先走步数 | slow 停在 |
|---|---|---|
| 倒数第 n 个节点 | n 步 | 倒数第 n 个 |
| 倒数第 n 个的前驱 | n+1 步 | 倒数第 n+1 个 |
| 链表中点 | 全程快 2 倍 | 中间节点 |
