# 25. K 个一组翻转链表

> 链接：https://leetcode.cn/problems/reverse-nodes-in-k-group/

## 题目描述

给你链表的头节点 `head`，每 `k` 个节点一组进行翻转，请你返回修改后的链表。

`k` 是一个正整数，它的值小于或等于链表的长度。如果节点总数不是 `k` 的整数倍，那么请将最后剩余的节点保持原有顺序。

> 不能只是单纯地改变节点内部的值，而是需要实际进行节点交换。

### 示例

```
输入：head = [1,2,3,4,5]，k = 2
输出：[2,1,4,3,5]
```

```
输入：head = [1,2,3,4,5]，k = 3
输出：[3,2,1,4,5]
```

约束：`1 <= k <= n <= 5000`，`0 <= Node.val <= 1000`。

## 思路推导

### 迭代法：找尾 + 反转子段 + 重接

每轮处理步骤：
1. 从 `pre.next` 出发走 `k` 步找到本组末尾 `tail`（如果不足 k 步则停止）。
2. 保存 `tail.next`（下一组的头）。
3. 断开本组：`tail.next = None`。
4. 反转本组（`[pre.next, tail]`），得到新头和新尾。
5. 重接：`pre.next = new_head`，`new_tail.next = next_group_head`。
6. `pre` 移动到 `new_tail`（本组翻转后的末尾），进入下一轮。

```
初始：dummy → [1 → 2 → 3] → [4 → 5]，k=3
       pre    head       tail next_head

反转 [1→2→3] 得到 [3→2→1]

重接：dummy → [3 → 2 → 1] → [4 → 5]
                         pre（移到 1）
```

### 找尾辅助函数

```python
def get_tail(node, k):
    for _ in range(k - 1):
        node = node.next
        if not node:
            return None
    return node
```

## Python 代码

### 迭代（推荐）

```python
class Solution:
    def reverseKGroup(self, head: Optional[ListNode], k: int) -> Optional[ListNode]:
        def reverse(head, tail):
            """反转 [head, tail] 这段链表，返回 (新头, 新尾)"""
            prev, curr = tail.next, head   # prev 初始为 tail.next（断点处）
            while prev is not tail:
                nxt = curr.next
                curr.next = prev
                prev = curr
                curr = nxt
            return tail, head              # 原尾变新头，原头变新尾

        dummy = ListNode(0, head)
        pre = dummy
        while True:
            # 找到本组的尾节点（从 pre.next 走 k-1 步）
            tail = pre
            for _ in range(k):
                tail = tail.next
                if not tail:
                    return dummy.next      # 不足 k 个，结束
            next_head = tail.next
            # 反转并重接
            new_head, new_tail = reverse(pre.next, tail)
            pre.next = new_head
            new_tail.next = next_head
            pre = new_tail
```

### 递归（简洁但空间 O(n/k)）

```python
class Solution:
    def reverseKGroup(self, head: Optional[ListNode], k: int) -> Optional[ListNode]:
        # 检查是否有 k 个节点
        curr, count = head, 0
        while curr and count < k:
            curr = curr.next
            count += 1
        if count < k:
            return head
        # 反转前 k 个
        prev, node = None, head
        for _ in range(k):
            nxt = node.next
            node.next = prev
            prev = node
            node = nxt
        # head 现在是本组尾节点，连接后续
        head.next = self.reverseKGroup(node, k)
        return prev   # prev 是本组新头
```

## 易错点

1. **反转函数的 `prev` 初始值**：迭代写法里 `reverse(head, tail)` 中 `prev = tail.next`（断点处，不是 None），反转到 `prev is tail` 停止，保证 tail.next 指向 tail.next 原有内容（即下一组头节点），随后再被 `new_tail.next = next_head` 覆盖。
2. **先保存 `next_head = tail.next`**：反转后 tail.next 被修改，必须提前保存。
3. **找尾时走 k 步（包括 pre.next 本身）**：有的写法走 k-1 步，要看起点是 pre 还是 pre.next，不要搞混。
4. **`pre = new_tail`**：new_tail 是反转后本组的末尾（即反转前的头节点），下一轮从这里开始找下一组。

### 反转子段的三种写法

```python
# 写法 A：标准反转，prev 初始为 None（反转整个链表用）
prev, curr = None, head
while curr:
    nxt = curr.next; curr.next = prev; prev = curr; curr = nxt

# 写法 B：反转到 tail，prev 初始为 tail.next（反转子段用）
prev, curr = tail.next, head
while prev is not tail:
    nxt = curr.next; curr.next = prev; prev = curr; curr = nxt
# 循环结束时 prev = tail（原尾变新头），curr = tail.next
```
