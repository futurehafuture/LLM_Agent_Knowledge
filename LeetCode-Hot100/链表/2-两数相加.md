# 2. 两数相加

> 链接：https://leetcode.cn/problems/add-two-numbers/

## 题目描述

给你两个**非空**的链表，表示两个非负的整数。它们每位数字都是按照**逆序**方式存储的，并且每个节点只能存储**一位**数字。

请你将两个数相加，并以相同形式返回一个表示和的链表。

> 你可以假设除了数字 0 之外，这两个数都不会以零开头。

### 示例

```
输入：l1 = [2,4,3]，l2 = [5,6,4]
输出：[7,0,8]
解释：342 + 465 = 807
```

```
输入：l1 = [0]，l2 = [0]
输出：[0]
```

```
输入：l1 = [9,9,9,9,9,9,9]，l2 = [9,9,9,9]
输出：[8,9,9,9,0,0,0,1]
```

约束：链表节点数在 `[1, 100]` 范围内，`0 <= Node.val <= 9`，数字无前导零（0 本身除外）。

---

## 思路推导

逆序存储正好符合小学加法从最低位开始的运算顺序，不需要翻转。

模拟竖式加法：
1. 同时遍历 `l1` 和 `l2`，每步取两个节点的值（若某链表已结束则取 0）。
2. 加上进位 `carry`，当前位结果 `s = v1 + v2 + carry`。
3. 新节点值为 `s % 10`，新进位为 `s // 10`。
4. 任意一个链表还有节点，或者进位不为 0，就继续循环。

---

## Python 代码

### 写法一：同时遍历（推荐）

```python
class Solution:
    def addTwoNumbers(self, l1: Optional[ListNode], l2: Optional[ListNode]) -> Optional[ListNode]:
        dummy = ListNode(0)
        curr = dummy
        carry = 0
        while l1 or l2 or carry:
            v1 = l1.val if l1 else 0
            v2 = l2.val if l2 else 0
            s = v1 + v2 + carry
            carry = s // 10
            curr.next = ListNode(s % 10)
            curr = curr.next
            if l1: l1 = l1.next
            if l2: l2 = l2.next
        return dummy.next
```

---

## 复杂度分析

| 指标 | 值 |
|---|---|
| 时间 | O(max(m, n))，m、n 分别为两链表长度 |
| 空间 | O(max(m, n))，结果链表长度为 max(m, n) + 1（有进位时） |

---

## 易错点

1. **循环条件是 `l1 or l2 or carry`**：三个都要判，缺任何一个都会漏掉最高位进位（如 `[5]+[5]` 应输出 `[0,1]`，如果只判 `l1 or l2` 就不会产生末尾的 `1`）。
2. **提前取值 `v1 = l1.val if l1 else 0`**：不能直接写 `l1.val`，因为 l1 可能已经是 None。
3. **最后推进指针 `if l1: l1 = l1.next`**：不能无条件 `l1 = l1.next`，否则对 None 调用 `.next` 报错。

## 知识点补充

### 正序存储版本（LeetCode 445）

如果是正序存储，不能直接从头遍历，需要先翻转或用**栈**辅助（从栈顶取最低位）。Hot 100 考的是逆序版，但面试可能追问正序版：

```python
# 445. 两数相加 II（正序存储）
def addTwoNumbers(self, l1, l2):
    s1, s2 = [], []
    while l1: s1.append(l1.val); l1 = l1.next
    while l2: s2.append(l2.val); l2 = l2.next
    carry = 0
    dummy = ListNode(0)
    while s1 or s2 or carry:
        v1 = s1.pop() if s1 else 0
        v2 = s2.pop() if s2 else 0
        s = v1 + v2 + carry
        carry = s // 10
        node = ListNode(s % 10)
        node.next = dummy.next
        dummy.next = node    # 头插法，保证最终是正序
    return dummy.next
```
