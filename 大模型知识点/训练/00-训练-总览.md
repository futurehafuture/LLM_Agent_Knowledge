# 训练 · 总览

```
损失函数 → 优化器 → 精度 → 正则化 → 微调 → 对齐/RLHF
```

## 推荐阅读顺序

1. 先搞懂损失函数（01 交叉熵）——这是所有训练的基础
2. 再看优化器（02 AdamW）——知道梯度怎么变成参数更新
3. 精度（03）——理解显存和数值稳定性
4. 正则化（04 Dropout/weight decay）——与优化器配套的基础训练知识
5. 微调（05 LoRA）——大模型训练的工程基础
6. SFT vs RL（06）——进入对齐阶段的概念桥梁
7. 策略梯度基础（07）——RL 训练的数学地基（不懂 RL 就先看这个）
8. RLHF 对齐（08 → 09）——从 PPO 到 GSPO 的完整演进
