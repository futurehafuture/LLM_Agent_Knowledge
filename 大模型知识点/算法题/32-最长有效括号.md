# LeetCode 32. 最长有效括号

> 给你一个只包含 `'('` 和 `')'` 的字符串，找出最长有效（格式正确且连续）括号子串的长度。
>
> - 输入: `"(()"` → 输出: `2`（最长有效子串 `"()"`）
> - 输入: `")()())"` → 输出: `4`（最长有效子串 `"()()"`）
> - 输入: `""` → 输出: `0`

---

## 0. 面试怎么答（先讲思路再上代码）

> "这题是 hard 但有三种经典解法：
>
> **思路 1（最推荐）：栈** —— 栈底永远存"最后一个没被匹配的右括号或起始 -1 哨兵"。遇到 `(` 入栈下标，遇到 `)` 弹栈，如果栈空再把当前下标入栈当新边界，否则用当前下标减栈顶下标更新答案。**时间 O(n)、空间 O(n)**。
>
> **思路 2：动态规划** —— $`dp[i]`$ 表示以 $`i`$ 结尾的最长有效括号长度。`(` 结尾必为 0；`)` 结尾要分两种情况：前一个是 `(` 直接 +2；前一个是 `)` 找它对应的左括号位置，能匹配上则 +2 再加该位置前一段。**时间 O(n)、空间 O(n)**。
>
> **思路 3：双向扫描计数器** —— 从左到右扫一遍，分别记 left/right 计数器，相等时更新答案，right > left 时清零。再从右到左对称扫一遍。**时间 O(n)、空间 O(1)**，省空间但思路稍绕。
>
> 我会先写栈解法（最直观稳）。"

## 1. 解法一：栈（最推荐）

### 1.1 核心思路

栈维护**"已经形成有效子串前的边界位置"**：

- 初始往栈里 push `-1`（哨兵：第一个有效子串的"前一位"）
- 遇到 `(`：把下标 push 进栈（候选的左括号位置）
- 遇到 `)`：
  - **先 pop**（试图配对一个左括号）
  - pop 之后栈空 → 当前 `)` 没人配对 → 它成为新边界，把当前下标 push 进栈
  - pop 之后栈非空 → 当前位置 - 栈顶 = 这段有效长度，更新答案

> **栈底永远是"上一个未匹配的 `)` 的位置"或哨兵 `-1`**。栈中除栈底外的元素都是未匹配的 `(` 的位置。

### 1.2 例子 walk-through

`s = ")()())"`，遍历过程：

| i | ch | 操作 | 栈 | max_len |
|---|---|---|---|---|
| 初始 | | push -1 | [-1] | 0 |
| 0 | `)` | pop -1 → 栈空 → push 0 | [0] | 0 |
| 1 | `(` | push 1 | [0, 1] | 0 |
| 2 | `)` | pop 1 → 栈顶 = 0 → 2 - 0 = 2 | [0] | 2 |
| 3 | `(` | push 3 | [0, 3] | 2 |
| 4 | `)` | pop 3 → 栈顶 = 0 → 4 - 0 = 4 | [0] | **4** |
| 5 | `)` | pop 0 → 栈空 → push 5 | [5] | 4 |

最终答案：4 ✅

### 1.3 代码

```python
class Solution:
    def longestValidParentheses(self, s: str) -> int:
        stack = [-1]   # 哨兵：当前段的左边界
        max_len = 0

        for i, ch in enumerate(s):
            if ch == '(':
                stack.append(i)
            else:  # ch == ')'
                stack.pop()
                if not stack:
                    # 当前 ')' 没人配对，作为新边界
                    stack.append(i)
                else:
                    # 当前段长度 = i - 栈顶
                    max_len = max(max_len, i - stack[-1])

        return max_len
```

复杂度：
- 时间 $`O(n)`$
- 空间 $`O(n)`$

## 2. 解法二：动态规划

### 2.1 状态定义

$`dp[i]`$ = **以下标 $`i`$ 结尾的最长有效括号子串长度**。

注意 "以 $`i`$ 结尾" 这个约束：意味着 $`s[i]`$ 必须是 `)`，且必须能形成有效子串。

### 2.2 状态转移

**Case 1：$`s[i] = `$ `(`**

不可能以 `(` 结尾形成有效串 → $`dp[i] = 0`$

**Case 2：$`s[i] = `$ `)` 且 $`s[i-1] = `$ `(`**

形如 `...()`，最后两个直接配对：

```math
dp[i] = dp[i-2] + 2 \quad (\text{若 } i \ge 2,\text{ 否则 } 2)
```

**Case 3：$`s[i] = `$ `)` 且 $`s[i-1] = `$ `)`**

形如 `...))`，要找 $`i-1`$ 处那段配完之后**再往前一位**：

```
i-1 处那段长度 = dp[i-1]
对应起始位置 = i - 1 - dp[i-1] + 1 - 1 = i - dp[i-1] - 1
```

如果 $`s[i - dp[i-1] - 1] = `$ `(`，则它和当前 `)` 配对：

```math
dp[i] = dp[i-1] + 2 + dp[i - dp[i-1] - 2]
```

（最后一项是再往前那段有效串，可能为 0）

### 2.3 例子

`s = "()(())"`：

| i | s[i] | dp[i] | 推理 |
|---|---|---|---|
| 0 | `(` | 0 | |
| 1 | `)` | 2 | s[0..1] = "()", dp[1] = dp[-1] + 2 = 2 |
| 2 | `(` | 0 | |
| 3 | `(` | 0 | |
| 4 | `)` | 2 | s[3..4] = "()", dp[4] = dp[2] + 2 = 2 |
| 5 | `)` | 6 | s[5] = ')', s[4] = ')': 找 s[5 - dp[4] - 1] = s[2] = '(' 匹配，dp[5] = dp[4] + 2 + dp[1] = 2 + 2 + 2 = 6 |

### 2.4 代码

```python
class Solution:
    def longestValidParentheses(self, s: str) -> int:
        n = len(s)
        if n < 2:
            return 0

        dp = [0] * n
        max_len = 0

        for i in range(1, n):
            if s[i] == ')':
                if s[i - 1] == '(':
                    # 形如 "...()"
                    dp[i] = (dp[i - 2] if i >= 2 else 0) + 2
                elif i - dp[i - 1] - 1 >= 0 and s[i - dp[i - 1] - 1] == '(':
                    # 形如 "...(())"
                    dp[i] = dp[i - 1] + 2
                    if i - dp[i - 1] - 2 >= 0:
                        dp[i] += dp[i - dp[i - 1] - 2]
                max_len = max(max_len, dp[i])

        return max_len
```

复杂度：
- 时间 $`O(n)`$
- 空间 $`O(n)`$

## 3. 解法三：双向扫描（O(1) 空间）

### 3.1 核心思路

只用两个计数器 `left`（已遇到的左括号数）和 `right`（已遇到的右括号数）：

**从左到右扫**：
- 遇 `(` → `left++`
- 遇 `)` → `right++`
- 若 `left == right`：当前是有效段，长度 `2 * right`，更新答案
- 若 `right > left`：永远配不上了 → 清零

**问题**：只从左到右扫漏掉 `((` 这种**永远凑不齐**但 `left` 也没超的情况。

**解决**：再**从右到左扫一遍**，对称处理：
- 遇 `)` → `right++`
- 遇 `(` → `left++`
- 若 `left == right`：更新
- 若 `left > right`：清零

### 3.2 例子

`s = "(()"` 单向扫描看不出来：
- 从左到右：`(` left=1, `(` left=2, `)` right=1, **left ≠ right 但没清零** → 永远拿不到 max_len
- 从右到左：`)` right=1, `(` left=1, **left == right → max_len = 2** ✅

### 3.3 代码

```python
class Solution:
    def longestValidParentheses(self, s: str) -> int:
        max_len = 0
        n = len(s)

        # 从左到右
        left = right = 0
        for ch in s:
            if ch == '(':
                left += 1
            else:
                right += 1
            if left == right:
                max_len = max(max_len, 2 * right)
            elif right > left:
                left = right = 0

        # 从右到左
        left = right = 0
        for ch in reversed(s):
            if ch == '(':
                left += 1
            else:
                right += 1
            if left == right:
                max_len = max(max_len, 2 * left)
            elif left > right:
                left = right = 0

        return max_len
```

复杂度：
- 时间 $`O(n)`$（扫两遍）
- 空间 $`O(1)`$ ✅

## 5. 常见错误 / 坑

### 5.1 栈解法不放哨兵 -1

```python
# 错误版
stack = []   # 没有哨兵
# 遇到 ) 时，弹栈后用 i - stack[-1] 会出现 i - (-1) 失效
```

哨兵 -1 让"以下标 0 开头的有效串"能正确算出 `i - (-1) = i + 1`。

### 5.2 栈里存什么

存**下标**，不是存字符。否则没法算长度。

### 5.3 DP 解法 Case 3 越界

```python
if i - dp[i - 1] - 1 >= 0 and s[i - dp[i - 1] - 1] == '(':
```

必须先判越界再访问，否则 `s[-x]` 在 Python 是反向访问会出错。

### 5.4 双向扫描忘了对称

只从左到右过不了 `"(("`，必须右到左也扫一遍。

## 7. 面试高频追问

### Q1: 三种解法你最推荐哪种？
**栈**。代码量适中、最不容易写错、思路清晰。DP 容易在 Case 3 的下标计算上出 bug。

### Q2: 栈的哨兵 -1 为什么必要？
为了让"从 0 开始的有效段"的长度可以用 `i - stack[-1]` 统一计算。否则需要特判第一段。

### Q3: 双向扫描为什么必须扫两遍？
单向扫描的清零条件是"右多于左"，但形如 `(((()` 永远不会触发清零，导致 `left ≠ right` 一直没匹配上。反向扫描时左右身份对换，能补上这种情况。

### Q4: 能不能用一遍 + 栈做到 O(1) 空间？
不能。**栈解法的空间下界是 $`\Theta(n)`$**（极端情况 `((((((` 栈会满）。要达到 O(1) 空间必须放弃栈，用双向扫描。

### Q5: 如果是多类型括号（`(){}[]`）怎么改？
DP 思路失效（无法只看相邻位置），双向扫描失效（多种括号不能简单计数）。**只能用栈**：栈里存 `(下标, 字符)`，配对时判断字符匹配。

### Q6: 如果允许"翻转一个字符"求最长有效，怎么做？
更复杂的 DP：枚举翻转位置，但 $`O(n^2)`$ 太慢。可以预处理"以 i 结尾的最长" + "以 i 开头的最长"，再 $`O(n)`$ 枚举翻转点。

## 8. 完整可运行测试

```python
def test():
    sol = Solution()
    cases = [
        ("", 0),
        ("(", 0),
        (")", 0),
        ("()", 2),
        ("(()", 2),
        (")()())", 4),
        ("()(())", 6),
        ("((((((", 0),
        (")))()))", 2),
        ("()()", 4),
    ]
    for s, expected in cases:
        got = sol.longestValidParentheses(s)
        status = "✅" if got == expected else "❌"
        print(f"{status} '{s}' -> {got} (expected {expected})")

test()
```
